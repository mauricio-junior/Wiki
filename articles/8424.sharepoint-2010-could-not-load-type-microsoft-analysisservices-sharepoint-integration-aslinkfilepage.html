To create a Power View report, first we need to create a BI Semantic Model Connection:

 

 

 

But If we did not deploy PowerPivot Web Application to the SharePoint application, we will get the following error 
Error: Could not load type 'Microsoft.AnalysisServices.Sharepoint.Integration.ASLinkFilePage'.


 

 




To solve the error 
  1. Open Central Administration
  2. System Settings
  3. Manage Farm Solutions

 

 



Here, select the powerpivotwebapp.wsp and deploy it to the SharePoint application.
