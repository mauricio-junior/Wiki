Background:

Following the upgrade of an existing SharePoint farm, the SharePoint Central Administration Upgrade Status page shows a Failed status. The status page contains the name and location of the SharePoint upgrade log file that you can view for more information. The SharePoint upgrade log will contain an error message that resembles the following:

[psconfigui] [SPSiteWssSequence2] [INFO] [1/21/2010 10:50:50 PM]: SPSite Url=http://davesellvmlhr2:46790

[psconfigui] [SPSiteWssSequence2] [WARNING] [1/21/2010 10:50:50 PM]: Feature could not be upgraded. Exception: Feature definition id '367b94a9-4a15-42ba-b4a2-32420363e018' could not be found.

[OWSTIMER] [SPContentDatabaseSequence] [INFO] [1/21/2010 10:51:57 PM]: SPContentDatabase Name=WSS_Content

[OWSTIMER] [SPContentDatabaseSequence] [ERROR] [1/21/2010 10:51:57 PM]: Found a missing feature Id = [367b94a9-4a15-42ba-b4a2-32420363e018]

[OWSTIMER] [SPContentDatabaseSequence] [INFO] [1/21/2010 10:51:57 PM]: SPContentDatabase Name=WSS_Content

[OWSTIMER] [SPContentDatabaseSequence] [ERROR] [1/21/2010 10:51:57 PM]: The feature with Id 367b94a9-4a15-42ba-b4a2-32420363e018 is referenced in the database [WSS_Content], but is not installed on the current farm. The missing feature may cause upgrade to fail. Please install any solution which contains the feature and restart upgrade if necessary.

Workarounds:

To work around this issue for a site collection:

  1. From Site Settings, in the Site Collection Administration section, select Site Collection Features.
  2. Activate the Report Server Integration Feature.

To work around this issue for multiple site collections:

  1. From SharePoint Central Administration select General Application Settings.
  2. In the Reporting Services section, select Reporting Services Integration.
  3. Activate the Reporting Services Integration feature for specific site collections or all site collections.

Applies to:

  • SharePoint Foundation 2010
  • SharePoint Server 2010
  • Microsoft SQL Server 2008 R2 Reporting Services Add-in for SharePoint 2010
