Introdução

​ No Windows 8 as vezes pode existir um pequeno problema na hora da ativação automatica do Sistema operacional, abaixo vamos mostrar como ativar o mesmo via linha de comando.

Ativando

1° Primeiramente acesse o Command Prompt  , clicando em Windows + R e digitando CMD e teclando ENTER.



2° Agora com o Prompt Aberto Digite: slmgr.vbs -ipk XXXXX-XXXXX-XXXXX-XXXXX-XXXXX. Substituindo o X pelo serial e tecle ENTER.



3° Pronto seu Windows 8 Esta Ativado, agora é só reiniciar seu PC.

Conclusão

 
 O procedimento acima mostra como ativar via linha de comando qualquer versão do Windows 8 via linha de commando.
