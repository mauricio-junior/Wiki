Microsoft beberapa waktu ini secara resmi telah merilis Windows 8.1. Ini merupakan edisi pembaharuan untuk Windows sebelumnya, yakni Windows 8. Pembaharuan ini dilakukan karena terdapatnya banyak kekurangan-kekurangan pada edisi sebelumnya. Bagi pengguna Windows 8, Microsoft memberikan sebuah kejutan. Mereka yang sudah menggunakan Windows 8, diberikan kemudahan untuk upgrade ke Windows 8.1 secara gratis yang bisa didapatkan di Windows Store.
Masihkah Windows 8.1 mengusung tampilan Metro UI ? Tentunya masih. Lantas, apakah hal yang baru pada Windows 8.1? Ada 8 hal baru yang bisa pengguna temukan pada edisi ini. Apa saja ? Semua hal tersebut akan saya bahas pada artikel berikut ini.

Internet Explorer 11
Intenet Explorer menjadi web browser bawaan yang tersedia pada semua versi Windows. Tentunya tidak terkecuali dengan edisi Windows 8.1. Berbeda dengan Windows 8 yang menggunakan Internet Explorer 10. Windows 8.1 sudah mulai menggunakan Internet Explorer 11. Apakah perbedaannya ? Apakah hanya versi semata? Tidak! IE 11 menggunakan memori dengan jumlah yang lebih sedikit dari sebelumnya. Selain itu, IE 11 bisa menjadi kabar bahagia bagi pengembang web. Mengapa? Karena IE 11 mulai mendukung tampilan web dengan edisi CSS terbaru.

Kembalinya Start Button
Sebelumnya, Windows memang sempat kehilangan ‘Start Button’. Namun kini nampaknya Microsoft mulai mengembalikan adanya Start Button. Komponen start button ini bila diklik akan mengarahkan Anda ke tampilan Start Screen Metro.

Ukuran Tile yang beragam
Pernahkah anda mengubah ukuran tile pada Start Screen? Bila pernah, maka Anda akan menemukan beberapa pilihan ukuran. Pada edisi sebelumnya, hanya disediakan dua pilihan ukuran yakni besar dan kecil. Pada Windows 8.1 ukuran tile mulai ditambahkan menjadi ukuran besar, lebar, menengah dan kecil. Jadi anda tidak akan bosan untuk mengotak-atik tile tersebut. Menjadikan Start Screen anda lebih menarik.

Tampilan Windows Store yang baru
Windows Store adalah tempat dimana pengguna bebas mengunduh ataupun membeli aplikasi atau games yang mereka sukai. Kini Windows Store memiliki tampilan yang lebih menarik. Dilengkapi dengan navigasi yang jelas.

Personalisasi yang menarik
Ekspresikan dirimu dengan beraneka pilihan tema Start Screen. Personalisasi pada edisi ini berbeda dengan yang ada pada Windows 8. Anda bisa menentukan warna latar belakang dan juga bisa menentukan warna aksen yang anda favoritkan. Selain itu anda juga bisa menggunakan latar belakang desktop sebagai latar belakang.

Fitur Proxy
Sebelumnya, banyak pengguna yang mengeluhkan aplikasi metro yang banyak mengalami masalah karena faktor koneksi. Pada pembaharuan Windows 8.1 ini sudah dilengkapi dengan fitur baru yang memungkinkan anda menggunakan sistem proxy untuk membenahi koneksi internet anda. Tentunya agar aktivitas anda dalam browsing di Windows Store menjadi lancar.

Lockscreen Dinamis
Lockscreen pada Windows 8 terlihat membosankan. Mengapa? Karena pengguna hanya diijinkan menggunakan satu gambar statis saja. Windows 8.1 mulai berubah. Anda bisa menikmati lockscreen dengan gambar slide menarik.

Booting Langsung ke Desktop
Pengguna sempat dibuat sebal saat era Windows 8. Setelah booting pengguna harus menunggu agar start screen tampil terlebih dahulu. Ini tentu membuat kesan komputer menjadi lambat. Karena pada saat startup banyak program aplikasi yang masih mempersiapkan sistemnya. Maka dari itu sistem Windows mengalami kesibukan. Pada Windows 8.1 ini, pengguna diberikan opsi untuk boot langsung ke desktop tanpa melalui start screen.
 
Bahasan tadi merupakan beberapa hal yang baru yang bisa anda temukan pada Windows 8.1.  Memang tidak ada perubahan yang signifikan bila dibandingkan dengan Windows 8 sebelumnya. Namun setidaknya ini lebih baik dari edisi sebelumnya. Semoga apa yang saya tulis disini bermanfaat bagi anda. Akhir kata saya ucapkan sekian dan terimakasih.
