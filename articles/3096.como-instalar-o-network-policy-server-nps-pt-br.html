Você pode utilizar este procedimento para instalar o Network Policy Server (NPS) através do Assistente para instalar Roles. O NPS é um serviço do role Network Policy and Access Services.

Para instalar O NPS, siga o passo a passo abaixo:

Passo 1 – Acesse o Server Manager, na opção Roles, clique em Add Roles…

Como Instalar o Network Policy Server (NPS) (pt-BR)

Passo 2 – Em Before You Begin, clique em Next. Em Select Server Roles, marque o checkbox Network Policy and Access Services e clique em Next.

Passo 3 – Na tela de Introdução, clique em Next e vá para a tela select Role Services e marque o checkbox Network Policy Server. Clique em Next.

Passo 4 – Para finalizar a instalação, clique Install, depois de revisar as opções selecionadas.

Pronto. Agora é só configurar o NPS.
