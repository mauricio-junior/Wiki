To Restore Your Target “Cloned” Server
1.            On your Skeleton server, Click Start>> Programs>> Accessories>>System Tools>> Backup.
 
Click on the Restore Wizard Button.
Click Next
Click Import File Button.
Browse to your system backup file.
This is the cloned server.
File Is Selected.
Click Ok
Make sure that the System State is checked, Click Next.
Click Advanced.
Click Next
Select Replace The File On Disk Only If It’s Older Than The Backup Copy, Click Next.
Take the defaults, Click Next.
Click Finish.
Restore Starts.
Now Restart Your VM Cloned PC.
Remember this is now a copy of the source PC, so IP addresses Computer names and such are duplicated on your network.
It would be wise to shut down the source before you restart your new VM  cloned PC.
