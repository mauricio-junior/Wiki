Introdução

  • O Microsoft Developer Network (MSDN) é a porção da Microsoft responsável por gerenciar o relacionamento da empresa com os desenvolvedores e testadores: desenvolvedores de hardware interessados ​​no sistema operacional (OS), os desenvolvedores sobre as várias plataformas OS, os desenvolvedores que utilizam a API e linguagem de scripts de aplicações da Microsoft. 

Fóruns MSDN

  • Fóruns MSDN são os fóruns baseados na web utilizado pela comunidade para discutir uma ampla variedade de tópicos de desenvolvimento de software.

Galeria do MSDN

  • A Galeria do MSDN é um repositório de autoria da comunidade com exemplos de códigos e projetos. Novo em 2008, o objetivo do site ainda está evoluindo para complementar o Codeplex, o site do projeto open-source de hospedagem da Microsoft.
Biblioteca do MSDN

A Biblioteca do MSDN é o repositório centralizado de documentação relacionados ao desenvolvedor oficial. Conjuntos de documentos são publicados por várias organizações de assistência do usuário dentro da Microsoft e a comunidade tem a oportunidade em muitas das páginas adicionar sua próprias anotações. O Conteúdo da Comunidade com as contribuições podem ser editados por qualquer pessoa. Desde maio de 2006, a Microsoft também começou a produzir uma versão em CD comunidade do MSDN Library disponíveis gratuitamente para download como imagens ISO em seu Download Center que contém apenas o conteúdo atualizado.

A distribuição comunitária do MSDN Library é destinado a desenvolvedores em países onde as conexões de internet banda larga não estão prontamente disponíveis.

Visual Studio Express Editions integrar apenas com a Biblioteca MSDN Express que é um subconjunto da Biblioteca MSDN completo, embora seja edição MSDN pode ser baixado gratuitamente e instalado standalone.

Em Visual Studio 2010 MSDN Library é substitm muitas das páginas adicionar sua próprias anotações. O Conteúdo da Comunidade com as contribuições podem ser editados por qualquer pessoa. Desde maio de 2006, a Microsoft também começou a produzir uma versão em CD comunidade do MSDN Library disponíveis gratuitamente para download como imagens ISO em seu Download Center que contém apenas o conteúdo atualizado.

A distribuição comunitária do MSDN Library é destinado a desenvolvedores em países onde asuído pelo novo Sistema de Ajuda, que é instalado como parte da instalação do Visual Studio 2010. Ajuda do aplicativo Library Manager é utilizado para instalar a Ajuda livros de conteúdo que abrange temas selecionados.

MSDN Magazine


A Microsoft fornece o conteúdo editorial para o MSDN Magazine, uma publicação mensal. A revista foi criada como uma fusão entre o Microsoft Systems Journal (MSJ) e Internet Microsoft Developer (MIND) revistas em Março de 2000. Edições anteriores do MSJ estão disponíveis online. O MSDN Magazine está disponível como uma revista impressa nos Estados Unidos e on-line em vários idiomas.
