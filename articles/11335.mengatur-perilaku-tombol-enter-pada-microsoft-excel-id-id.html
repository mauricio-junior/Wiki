Dalam kondisi default, menekan tombol Enter pada aplikasi Microsoft Excel akan menyebabkan sel terpilih berpindah kebawah. Perilaku penekanan tombol Enter ini sebetulnya bisa kita rubah dengan mudah. Misalkan saja jika Anda ingin agar sel terpilih berpindah ke-kanan setelah menekan tombol Enter. Caranya sebagai berikut.

Klik tab File --> Options untuk menampilkan jendela Excel Options. Berikutnya pilih Advanced dibagian kiri. Lalu dibagian kanan, didalam bagian Editing Options, Anda dapat mengatur perilaku tombol Enter ini dibagian After pressing Enter, move selection. Tentunya ada 4 pilihan arah yang disediakan, yaitu Up; Down; Left; dan Right.

Jika diinginkan, Anda bahkan dapat menonaktifkan perilaku pergeseran sel terpilih ini, yaitu dengan menghilangkan tanda ceklist di bagian After pressing Enter, move selection tersebut. Dan setelah dinonaktifkan, jika Anda menekan tombol Enter, maka sel terpilih tidak akan berpindah dan tetap diam di sel semula.
