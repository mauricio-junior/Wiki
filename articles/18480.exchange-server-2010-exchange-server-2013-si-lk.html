Exchange Server 2013 සකසුරුවම් කීරීම සඳහා මා භාවිතා කරනු ලබන්නේ Microsoft සමාගමේ තවත් නවතම නිකුත් කීරීමක් වන Windows 2012 මෘදුකාංගයි.

මුලින්ම Windows 2012 සර්වරය සියලුම නවතම updates සමගින් යාවත්කාලීන කිරීම සිදුකලයුතු වනවා.

ඉන් අනතුරුව එම Windows 2012 සර්වරය අපගේ Active Directory හා සම්බන්ද කළයුතු වන අතර එම සම්බන්දතාවය සිදුකරන ආකාරය ගැන මෙම ලිපිය මගින් සඳහනක් කිරීමට බලාපොරොත්තු වන්නේ නැත.

ඉහත සඳහන් ආකාරයට Active Directory හා සම්බන්දවීමෙන් පසු, Active Directory හි පාලක වරප්‍රසාද (Administrative privileges) සහිත user account එකකින් login වීම සිදුකල යුතුයි.

ඉන් අනතුරුව Windows 2012 සර්වරයෙහි අඩංගු Windows Power-Shell මෙවලම ක්‍රියාත්මක කොට පහත සඳහන් නියෝගය ක්‍රියාත්මක කළයුතු වන අතර මෙම නියෝගය මගින් Exchange Server 2013 සඳහා අවශ්‍ය සියලුම පූර්වාවශ්‍ය (Pr-requisites) අංග සකසුරුවම් කීරීම සිදු වනවා.
