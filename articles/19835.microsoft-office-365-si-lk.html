Microsoft සමාගමේ නවතම පෙරළිකාර හඳුන්වාදීම වන Microsoft Office 365 මගින් කාර්ය්‍යාල තුල දෛනිකව සිදුකරනු ලබන කාර්යයන් වෙත අලුත් මුහුණුවරක් ලබා දී ඇත.

Microsoft Office 365 භාවිතයෙන්, ඔබ භාවිතා කිරීමට හුරු පුරුදු Microsoft Office හා බැඳුනු මෙවලම් ලෝකයේ ඕනෑම තැනක සිට අවශ්‍ය ඕනෑම වෙලාවක භාවිතා කිරීමේ පහසුකම ඇත.

මෙම සේවාව ලබා ගැනීමට අවශ්‍ය කරුණු සහ එම සේවාව සකසුරුවම් කරගැනීමේ පියවර පැහැදිලි කිරීම මෙම ලිපියෙහි අරමුණයි.

 පළමුව අප කලයුත්තේ  http://office.microsoft.com වෙබ් අඩවිය වෙත පිවිසීමයි.



එම වෙබ් අඩවියේ සඳහන් පරිදි, ව්‍යාපාරික​ අවශ්‍යතාවයන් සඳහා අප පිවිසිය යුතුවන්නේ මෙම වෙබ් අඩවියේ “See Options for business”  යන කොටසට වන අතර ඉන්පසු අපගේ තාවකාලික සේවාවන් පණ ගැන්වීමට සහ අපට අවශ්‍ය සේවා පැකේජය හඳුනා ගැනීමට  “Compare plans” මත click කිරීම සිදුකල යුතු වනවා.


අනතුරුව අපට අවශ්‍ය සේවා පැකේජය ලබා ගැනීමේදී, ආයතනයේ ප්‍රමාණය හා අවශ්‍ය සේවාවන් මත තෝරා ගැනීම සිදු කල හැකියි. 


අපගේ ලිපියේ සඳහන් කිරීම සඳහා Enterprise (E3) පැකේජය තෝරාගනු ලබන අතර, ඒ තුල අඩංගු අංග පිළිබඳව මෙම ලිපියෙන් සාකච්ඡා කරනු ලැබේ.


Enterprise (E3) පැකේජය තෝරාගනු ලබීමෙන් අනතුරුව සිදුකල යුතු වන්නේ අපගේ විස්තර ලබාදීම තුලින් අපගේ d-image.ashx/__size/550x0/__key/communityserver-wikis-components-files/00-00-00-00-05/86257.4.png" style="border-width:0px;border-style:solid;" />

අපගේ ලිපියේ සඳහන් කිරීම සඳහා Enterprise (E3) පැකේජය තෝරාගනු ලබන අතර, ඒ තුල අඩංගු අංග පිළිබඳව මෙම ලිපියෙන් සාකච්ඡා කරනු ලැබේ.


Enterprise (E3) පැකේජය තෝරාගනු ලබීමෙන් අනතුරුව සිදුකල යුතු වන්නේ අපගේ විස්තර ලබාදීම තුලිනලියාපදිංචිය සම්පුර්ණ කිරීමේ කාර්යයයි.


මෙහි ඇති වැදගත් අංග වනුවේ, අපගේ රට ලෙස “ශ්‍රී ලංකාව”, අපගේ විද්‍යුත් තැපෑල ලිපිනය සහ අපගේ ජංගම දුරකථන අංකයක් ලබාදීමයි.

මෙයින් විද්‍යුත් තැපෑල ලිපිනය සහ අපගේ ජංගම දුරකථන අංකය වැදගත් වනුවේ, අපගේ රහස් අංකය නැතොත් Password එක​ අස්ථානගත වුවහොත්, නැවත එය ප්‍රතිෂ්ඨාපනය කිරීමට ඔබගේ විද්‍යුත් තැපෑල ලිපිනය සහ ඔබගේ  ජංගම දුරකථන අංකය අවශ්‍ය වන බැවිනි.

මෙහි අනෙකුත් වැදගත් කරුණ වන්නේ අප විසින් සාදනු ලබන User ID සහ රහස් අංකය මතකයේ තබා ගැනීමයි.

මෙම User ID එක පහත සඳහන් ආකාරයට සිදු කල හැකියි.



අපගේ ලියාපදිංචිය අවසාන වූ සැණින් ඔබට පහත සඳහන් ආකාරයට Microsoft Office 365 Admin center වෙත පිවිසිය හැකියි.



අනතුරුව අප කලයුතු ප්‍රධානතම​ කරුණ වන්නේ, ඔබ ආයතනයට අයත්වන Public Domain Name සඳහන් කිරීම සහ එය ක්‍රියාත්මක කරවා ගැනීමයි.

මේ සඳහා Microsoft Office 365 Admin center හි “Domain” අංගය ක්‍රියාත්මක කොට “Add a Domain” ක්‍රියාත්මක කලයුතු වනවා.



මෙහිදී ඔබගේ අනන්‍යතාවය තහවුරු කිරීම සහ Domain Name සඳහා හිමිකාරීත්වය තහවුරු කලයුතු වන අතර, මෙම ක්‍රියාවලිය කිසිදු තාක්ෂණික දැනුමක් නැති පුද්ගලයෙකුට වුවද ඉතා පහසුවෙන් කරගැනීමට ඉඩ සලසා ඇත.

මේ සඳහා පිවිසුමක් ලබා ගැනීමට Start Step 1 මත click කිරීම සිදු කල යුතුයි.


මෙහි දෙවන පියවර ලෙස ඔබට හිමිකාරීත්වයක් ඇති, ඔබගේ විද්‍යුත් පණිවුඩ හුවමාරුවට භාවිතා කිරීමට බලපොරොත්තුවන Public Domain Name සඳහන් කිරීම කළ යුතුයි.



ඉන්පසු ඔබගේ හිමිකාරීත්වය තහවුරු කිරීමට “Confirm ownership” මත click කිරීම සිදු කල යුතුයි.



මීළඟ පියවරෙන් ඔබගේ Public Domain Name ලියාපදිංචි කොට ඇති ආයතනය සඳහන් කලයුතු වන අතරම එමගින් එම එක් එක් ආයතනයට අනුකූල සකසුරුවම් කිරීමේ පිළිවෙත ඉතාමත් අගනා ආකාරයට සවිස්තරව සඳහන් කර ඇත. උදාහරණයක්‌ වශයෙන් Godaddy ආයතනය සඳහන් කිරීම සිදුකරනු ලබනවා.



ඉහතින් සඳහන් ආකාරයට පිළිවෙත් අනුගමනය කිරීමෙන් පසුව “Done, Verify now”  ක්‍රියාත්මක කිරීමෙන් අපගේ Public Domain Name සකසුරුවම් කිරීම සහ බල ගැන්වීම සිදුවේ.


මීළඟට ඔබ යම්කිසි ආයතනයක් මගින් සේවාවන් ලබාගන්නේ නම් එම ආයතනය පිළිබඳ සටහන් කිරීම සිදුකලයුතු වනවා. මෙම ක්‍රියාවලිය තුලින් ඔබ ආයතනයේ Microsoft Office 365 සේවාව යම් උවදුරකට ලක් වුවහොත් ඒ පිළිබඳව ඔබට සේවාවන් සපයන ආයතනයේ ඉංජිනේරුවරුන් දැනුවත් කිරීමක් සිදුකරනු ලබනවා. මෙමගින් ඔබට ඉතා උසස්‌ සේවාවක් ලබාගැනීමට මග පාදා ඇත.
