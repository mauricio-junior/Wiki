Windows Forms é um conjunto de classes no .NET Framework que permite aos desenvolvedores criar rapidamente aplicações Windows cliente, usando bibliotecas poderosas para controle com interface de usuário e exibição de informações. Em Windows Forms, um form é uma interface visual onde informação e controles são exibidos para um usuário final. Controles são botões, labels, gráficos e outros elementos de interface de usuário (UI) onde o usuário final pode interagir com o form. Quando um usuário interage com controles num form, como clicar em um botão, a ação gera um evento que é então manuseado pelo código em sua aplicação para executar tarefas, por exemplo, executando uma soma, ou abrindo uma pasta no sistema operacional, ou interagindo com outros Windows Forms. O controle padrão encontrado no visual studio para Windows Forms é também extensível com a classe Usercontrol, possibilitando aos desenvolvedores criar seus próprios controles únicos herdando propriedades e funções de um conjunto de controles existentes. Existem também várias companhias terceirizadas oferecendo conjunto de controles adicionais às bibliotecas extensíveis já disponíveis no Visual Studio.
