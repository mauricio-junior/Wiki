While troubleshooting email flow at times Administrators might require message details. The message details were easily available with the help of Microsoft Outlook Client, however not all messages sent from the client would be available. For example, internal message details won't be available and only emails sent to external recipients will be available.

In Exchange Server 2013 we have this option to read the Message Details via Outlook Web App.
