Visão Global:

O SharePoint Designer 2013, auxilia usuários avançados e desenvolvedores a criar rapidamente soluções para atender às necessidades de negócio das empresas. Com ele é possível desenvolver inúmeras soluções para diversos cenários comuns em nosso cotidiano sem o uso de código, seu fluxo de trabalho basicamente trabalha a partir de sites de colaboração onde são feitas as edições e as alterações realizadas são disponibilizadas aos usuários finais a partir da publicação do site na web. Entretanto, essas soluções atendem diretamente a linha de integração de dados, inteligência de negócios e fluxos de trabalho também conhecidos como Workflow. Todavia, ele aproveita os blocos de construção disponíveis a fim de facilitar o desenvolvimento e o reaproveitamento do ambiente e do desenvolvimento realizado. Assim, os desenvolvedores podem utilizar o SharePoint Designer 2013 para iniciar rapidamente seus projetos.


Procedimento de instalação

SharePoint Designer 2013 é a ferramenta escolhida para o rápido desenvolvimento de aplicativos do SharePoint 2013. Usando o SharePoint Designer, usuários avançados e desenvolvedores podem criar rapidamente soluções SharePoint em resposta às necessidades de negócios. Os usuários avançados podem compor sem código soluções que abrangem uma variedade de cenários comuns, a partir de sites de colaboração e publicação na web a linha de negócios de integração de dados, soluções de inteligência de negócios e fluxos de trabalho humanos, aproveitando os blocos de construção disponíveis no SharePoint em um fácil de usar ambiente.

Além disso, os desenvolvedores podem usar o SharePoint Designer 2013 para obter um início rápido em projetos de desenvolvimento do SharePoint.

Guia de instalação:
 

Faça o download da versão que você precisa, e clique no executável para extrair os arquivos.



Aceite os termos deste acordo, caso contrário, você não pode instalar.



Nas opções de instalação você pode escolher o que você quer instalar, eu pessoalmente deixá-lo como ele é "out-of-the-box ".



Você pode preencher o seu nome completo eo nome da organização.



Você verá o progresso da instalação .. Por favor, aguarde alguns instantes.



Basta clicar em Fechar quando a instalação estiver concluída.



Com o estilo Windows 8 é um pouco diferente, mas basta ir ao diferente, mas basta ir a "Iniciar" e clique em SharePoint Designer 2013.




O programa irá carregar.



E o SharePoint Designer 2013 é agora e pronto para ser usado em SharePoint 2013.
