Hi All,

If you have to extend your OnPrem Active Directory infrastructure to Azure Active Directory, Microsoft introduced a powerful tool (All-in-one) called "AADC or Azure AD Connect".After installing this Sync tool, the Configuration Wizard ask for a Synchronization option :
  • ADFS (Active Directory Federation Services) 
  • PHS (Password Hash Synchronization) 
  • PTA (Pass-Through Authentication) 

With or Without a Seamless SSO.

To help you taking the right decision, i created and published a design considerations guide with all informations you need to know about PHS, PTA or ADFS options. Pros & Cons of each sychronization options are also listed on this document.

This guide is available on Slideshare (for reading) and Microsoft TechNet Gallery (for download).
