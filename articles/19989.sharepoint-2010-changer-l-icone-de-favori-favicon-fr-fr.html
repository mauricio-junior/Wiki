L'icône de favori (le FavIcon), l'icône qui représente le site Web, est une partie importante de n'importe quelle page. Spécifiquement dans SharePoint, si cette icône est laissé par défaut il affichera l'icône orange vif. 

Pour maintenir l'image de marque de votre page Web via votre icône de favori , suivez la procédure ci-dessous :

  1. Téléchargez votre fichier favicon dans votre bibliothèque d'images
  2. Publiez et approuvez votre icône à travers les flux de travail de Publishing SharePoint
  3. Ouvrez votre page maître dans SharePoint Designer 2010
  4. Recherchez la balise < SharePoint:SPShortcutIcon > dans l'entête HTML
  5. Modifiez la balise « IconUrl » avec l'URL relative de votre favicon
  6. Sauvegardez et archivez votre page maître.
  7. Publiez et approuvez votre page maître à travers les flux de travail de Publishing SharePoint
