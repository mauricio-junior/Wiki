Create new C# or VB project using Trust Services SDK

Start with appropriate project type

Configure references paths:

  • Open project properties, switch to Reference Paths tab
  • Add BIN folder from the Trust Service SDK to reference paths, save project
    This >
  • Right click References node in project tree, select Add Reference
  • In Browse tab, navigate to the same directory, add following assemblies:

    •     Microsoft.SqlAzure.Trust.EdgeComplianceModule.dll
        Microsoft.SqlAzure.Trust.SqlEdgeComplianceModule.dll
        Microsoft.SqlAzure.Trust.TfsClientServerCommon.dll
        Microsoft.SqlAzure.Trust.WcfFederation.dll
        Security.Cryptography.dll

    Configure the references to copy them to output directory:

    • Select all these assemblies under References node,
    • Right click and select Properties
    • Set Copy Local property to True.
