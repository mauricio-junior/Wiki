This is an informational event and it specifies that the Microsoft Exchange Transport service successfully started.  It is logged every time the Microsoft Exchange Transport service starts.

Event Viewer Details

 

 Product   Microsoft Exchange 
 Version  15.0
 Event ID  1001
 Source  MSExchangeTransportService
 Message  The service has started successfully. PID XXXX.

 

Causes

This event is logged every time your Exchange server restarts or the Microsoft Exchange Transport service restarts.

Resolutions

This is an informational event that can safely be ignored.
