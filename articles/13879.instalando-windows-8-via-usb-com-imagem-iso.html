Introdução

O benefício em se instalar o Windows via USB é porque geralmente é muito mais rápido que a instalação via DVDs.

Antes de começar, você vai precisar de

• Windows 8 em arquivo ISO

• 4 GB USB no mínimo.

 A Microsoft tem um utilitário gratuito que automaticamente vai fazer uma drive de USB inicializável a partir de um arquivo  ISO do Windows 8.

Instalando Windows 8 via USB com Imagem ISO

 Primeiro, faça o download e instale o utilitário.

 Não se preocupe ele Funciona Tanto para Windows 7 como para o Windows 8.

Acesse e faça o Download: http://images2.store.microsoft.com/prod/clustera/framework/w7udt/1.0/en-us/Windows7-USB-DVD-tool.exe

Depois de Instalado Procure o Ícone na sua tela inicial do Windows 8.

Agora Selecione o arquivo ISO do Windows 8 e clique em Next “Avançar”.


 
Agora Conecte o Drive USB.

E Selecione USB device.

 

Agora Selecione a sua unidade USB na lista e clique em iniciar a cópia “Begin Copying.


 
Esse processo pode demorar alguns minutos, dependendo da velocidade do seu drive USB.

 

Finalizando e Só pegar seu drive USB e Fazer Boot no seu PC ou notebook para Instalar o Windows 8.

Conclusão

Nesse procedimento mostramos como Instalar o Windows 8 via USB com Imagem ISO.
