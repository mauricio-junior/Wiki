    • Before using the instructions in this extension, set up the remote access VPN test lab with the instructions found in Test Lab Guide: Demonstrate Remote Access VPNs.

    Step 1: Install the Connection Manager Administration Kit on APP1

    1. On APP1, run Server Manager.
    2. In the console tree, click Features. In the details pane, click Add Features.
    3. In the Features list, click Connection Manager Administration Kit, click Next, and then click Install.
    4. Click Close.

    Step 2: Use the Connection Manager Administration Kit to Create a Profile

    1. On APP1, click Start, point to Administrative Tools, and then click Connection Manager Administration Kit.
    2. On the Welcome page, click Next.
    3. On the Select the Target Operating System page, click Next.
    4. On the Create or Modify a Connection Manager profile page, click Next.
    5. On the Specify the Service Name and the File Name page, type Contoso VPN in Service name, type contoso in File name, and then click Next.
    6. On the Specify a Realm Name page, click Next.
    7. On the Merge Information from Other Profiles page, click Next.
    8. On the Add Support for VPN Connections page, click Phone book from this profile, in Always use the same VPN server, type 131.107.0.2, and then click Next.
    9. On the Create or Modify a VPN Entry page, click Next.
    10. On the Add a Custom Phone Book page, uncheck Automatically download phone book updates, and then click Next.
    11. On the Configure Dial-up Networking Entries page, click Next.
    12. On the Specify Routing Table Updates page, click Next.
    13. On the Configuration Proxy Settings for Internet Explorer page, click Next.
    14. On the Add Custom Actions page, click Next.
    15. On the Custom Logon Bitmap page, click Next.
    16. On the Display a Custom Phone Book Bitmap page, click Next.
    17. On the Display Custom Icons page, click Next.
    18. On the Include a Custom Help File page, click Next.
    19. On the Display Custom Support Information page, type 1-800-CONTOSO in Support information, and then click Next.
    20. On the Display a Custom License Agreement page, click Next.
    21. On the Install Additional Files with the Connection Manager profile page, click Next.
    22. On the Build the Connection Manager Profile and its Installation Program page, click Next.
    23. On the Complete page, click Finish.
    24. Click Start, and then click Computer.
    25. In Computer, open the C:\Program Files\CMAK\Profiles\Windows 7 and Windows Vista folder.
    26. Right-click the contoso folder, point to Share with, and then click Advanced Sharing.
    27. In contoso Properties, click the Advanced Sharing button, click Share this folderOn the Install Additional Files with the Cstrong>, click OK, and then click Close.

    Step 3: Demonstrate the Profile on CLIENT1

    1. Connect CLIENT1 to the Corpnet subnet.
    2. Click Start, type \\app1\contoso, and then press ENTER.
    3. In the contoso folder window, double-click the contoso application.
    4. In Contoso VPN, click Yes, and then click OK.
    5. In the Contoso VPN window, click Cancel.
    6. Connect CLIENT1 to the Internet subnet.
    7. In the Network Connections window, double-click the Contoso VPN connection.
    8. Type user1 in User name, the password in Password, and CORP in Domain.
    9. Click Connect. You should see a successful VPN connection, identifying itself as being on the corp.contoso.com network.
    10. From the Command Prompt window, type ping app1, and then press ENTER. You should see the name app1.corp.contoso.com resolved to the IPv4 address 10.0.0.3 and four successful replies.
