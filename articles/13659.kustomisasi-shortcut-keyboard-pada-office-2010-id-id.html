Shortcut keyboard atau yang umum disebut dengan Shortcut saja merupakan istilah untuk penekanan beberapa tombol tertentu pada keyboard untuk menjalankan perintah-perintah yang spesifik pada aplikasi Office. Beberapa shortcut yang umum digunakan misalnya seperti CTRL + S untuk menyimpan file atau CTRL + P untuk perintah pencetakan. Anda juga tentunya dapat membuat shortcut keyboard tertentu sesuai dengan pe/div>

Kustomisasi Shortcut rintah yang Anda inginkan. Pada Office 2010 caranya sebagai berikut.

Jalankan salah satu aplikasi Office 2010, sebagai contoh misalnya aplikasi Word. Lalu klik menu File > Options untuk menampilkan jendela Word Options. Di bagian kiri, klik pada pilihan Customize Ribbon. Di bagian kanan agak kebawah, klik tombol Customize di sebelah Keyboard Shortcuts.

Jendela Customize Keyboard akan ditampilkan. Pilih kategori perintah yang ingin Anda atur shortcut keyboardnya di bagian Categories dan pilih perintahnya secara spesifik di bagian Commands. Jika perintah yang Anda pilih sudah tersedia shortcutnya Anda dapat melihatnya di bagian Current keys. Untuk menggunakan shortcut keyboard baru yang Anda inginkan, klik di dalam bagian Press new shortcut key dan tekan kombinasi tombol keyboard sesuai keinginan Anda.

Klik tombol Assign untuk menerapkan shortcut keyboard baru tersebut dan klik tombol Close untuk menutup jendela Customize Keyboard.

Catatan: Jika shortcut keyboard yang Anda masukan sudah digunakan oleh perintah yang lain, Anda dapat melihatnya di bagian Currently assigned to. Dan dengan menggunakan shortcut baru tersebut maka shortcut untuk perintah sebelumnya otomatis akan dihapuskan.
