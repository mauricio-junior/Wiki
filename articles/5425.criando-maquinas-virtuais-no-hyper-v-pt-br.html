Introdução

Esse artigo demonstra como criar uma máquina virtual utilizando a ferramenta Hyper-V.

Aplica-se a:

  • Windows Server 2008;
  • Windows  Server 2008 R2.

Criando Máquinas Virtuais no Hyper-V

Abra o servidor (host) que tem a Feature do Hyper-V instalada e que suporta as máquinas virtuais.

1 – Abra o gerenciador do Hyper-V  e clique em Actions, fica na parte superior direita e depois seleciona New.

Abrindo New irá ter 3 opções e uma delas é criar Virtual Machine.

Clique em Virtual Machine como informado na Figura 1.1 e mais detalhado na figura 1.2

 

Figura 1.1

Figura 1.2

 

Após ter selecionado Virtual Machine, segue a tela com informações sobre a criação da máquina virtual.

 

 

 

 

Clique em Next

 

Essa tela nos mostra e nos pede para digitar um nome Amigável para sua nova máquina Virtual

E o local onde será armazenado sua VM, por padrão já é informado o caminho mostrado no print em Location, mas se quiser instalar em outro local, apenas clique no Box do nome: Store The Virtual Machine in a Different Location e clique em Browse para informar o local.

 

 

Clique em Next

 

Essa tela nos mostra e pede para informar qual vai ser a quantidade de memória Ram distribuida para a VM que está criando.

Esse valor pode ser alterado para mais ou para menos depois quando sua VM estiver pronta e com o SO já em execução.

Mas para isso é necessário desligar a VM, alterar e depois ligar novamente.

 


Clique em Next

 Aqui temos a questão que envolve a parte de Network do Hyper-V e consequentemente qual placa de rede será utilizada para sua VM ter conectividade com a rede, ou se apenas quiser que sua VM tenha conectividade com as VMS internas ou apenas conectividade com seu Host.

Isso tudo depende de como criou a rede no próprio gerenciador do Hyper-V

 

Clique em Next

 

Nesse print, informamos como será nosso VHD, se vai ser criado um novo, se já existe e você quer usar ( ex: template) ou por ultimo simplesmente usar um existente.

Issomargin:0cm 0cm 10pt;">No meu caso estou criando um novo de 80GB

 

Clique em Next

 

Aqui informamos de onde um novo S.O será instalado, nele podemos instalar depois, que seria a primeira opção e após essas opção, temos as que nos interessa, podemos visualizar que temos como instalar através de uma mídia de DVD e também através de um .ISO, que seria a imagem do S.O, selecione a que você necessitar, no meu caso estou instalando via DVD.

 

 

Clique em Next

 

Após terminar os passos anteriores com sucesso, nessa última tela antes de começar a instalação, nos informa um resumo do que será feito, verifique se está tudo correto.

 

 

Clique em Next

Aqui já vemos a VM criada e já sendo executada, no meu caso eu criei a VM de nome VSERVERIIS.

 

Conclusão


Nesse tutorial, expliquei uma forma rápida e básica de como se criar uma Máquina Virtual no Hyper-V com o Windows Server 2008 R2.
