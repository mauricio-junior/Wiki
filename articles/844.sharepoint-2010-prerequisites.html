The following software and upgrades are prerequisites for the SharePoint 2010 installation.

Application Server

  • Microsoft .NET Framework version 3.5 Service Pack 1 - Download here
  • Microsoft Chart Controls for the Microsoft .NET Framework 3.5 - Download here
  • Microsoft Sync Framework Runtime v1.0 (x64) - Download 1.0 here, Download 2.0 here
  • Microsoft Filter Pack 2.0 - Download here
  • Windows PowerShell 2.0 - Download here
    Required to install on Windows Vista and Windows Server 2008. Provided with Windows 7 and Windows Server 2008 R2.           
  • SQL Server 2008 Native Client - Download here
  • Microsoft SQL Server 2008 Analysis Services ADOMD.NET - Download here
  • ADO.NET Data Services version 1.5 CTP2 - Download here
  • Windows Identity Foundation (WIF) - Download here

SQL Server 2008

  • SQL Server 2008 Service Pack 1 - Download here
  • Cumulative update 2 or higher - Download CU2 here; latest CU is #7.
