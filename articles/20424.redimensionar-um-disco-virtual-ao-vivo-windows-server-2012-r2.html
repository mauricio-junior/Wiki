Introdução

Com o Windows Server 2012 R2 você pode redimensionar os discos virtuais ao vivo, mesmo que esteja sendo utilizados por máquinas virtuais. Para isso, basta que os serviços de integração estejam instalados na máquina virtual (versão R2) e o disco esteja na controladora SCSI.
