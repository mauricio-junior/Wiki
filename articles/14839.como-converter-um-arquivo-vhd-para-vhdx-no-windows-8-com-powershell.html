Introdução

O formato VHDX e encontrado no Windows 8 e no Windows Server 2012,  caso precise fazer a conversão de um arquivo VHD das versões mais antigas do Windows utilize o procedimento abaixo.

Usando PowerShell

Abra o PowerShell no Windows 8 Pressione Windows + X e em Executar e digite o comando abaixo Powershell .

Com o PowerShell aberto Digite o comando abaixo Onde d:\temp\VHD1.vhd e o seu diretório que está o seu VHD e d:\temp\VHD2.vhdx e o diretório onde vai ser criado o seu VHDX.

 

Convert-VHD -Path d:\temp\VHD1.vhd –DestinationPath d:\temp\VHD2.vhdx

 

Pronto Finalizando o comando, aguarde até finalizar e seu VHDX estará criado.
