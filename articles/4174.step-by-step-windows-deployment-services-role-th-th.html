Step By Step วิธีการติดตั้งและปรับแต่ง Windows Deployment Services Role

อ้างอิงจากการทำความรู้จัก WDS Role ในบทความที่แล้ว http://social.technet.microsoft.com/wiki/contents/articles/3847.aspx

ในบทความนี้เสนอขั้นตอนการติดตั้ง Windows Deployment Services และ Review การปรับแต่งค่าต่าง ๆ เบื้องต้นดังต่อไปนี้

ขั้นตอนการติดตั้ง Windows Deployment Services

1. ที่เครื่องมือ Server Manager เลือก Role Summary > Add Roles
2. ที่หน้าต่าง Add Roles Wizard เลือก Windows Deployment Services จากนั้นกด Next เพื่อติดตั้งต่อไป
3. ที่ตัวเลือกของการเพิ่ม Role Services ให้เลือกทั้ง Deployment Server และ Transport Server

หมายเหตุ: หลังการติดตั้ง Windows Deployment Services จะยังไม่ Start จนกว่าจะปรับแต่งเรียบร้อยแล้ว คุณสามารถตรวจสอบการติดตั้ง Server Roles ได้จาก Log ในเครื่อง Server ที่เก็บไว้ใน Folder ดังรายการต่อไปนี้
C:\Windows\Logs\CBS\cbs.txt
C:\Windows\Logs\ServerManager.txt
C:\Windows\Logs\ServerManagerInstallationLog.txt


ขั้นตอนในการปรับแต่ง Windows Deployment Services เบื้องต้น

1. หลังจาก Setup WDS Role เสร็จแล้วให้ทำการเรียก Windows Deployment Service Console ขึ้นมา และทำการคลิกขวาเลือก Configure Server



2. Wizard ที่ช่วยในการติดตั้งจะให้คุณเลือกพื้นที่สำหรับการเก็บ Share Folder เพื่อเป็นต้นฉบับสำหรับการ Deploy Image (แนะนำให้เลือกพื้นที่ Folder ที่อยู่คนละ Partition กับ System Drive)

3. จากนั้น Wizard จะให้เลือกการตอบรับในกรณีที่มีเครื่อง Client Boot จาก Network เข้ามาหาที่เครื่อง WDS Server ซึ่งมี 3 ประเภทให้เลือกคือ

  • Do not respond to any client computers = ไม่ตอบรับการร้องขอเพื่อเชื่อมต่อกับ WDS Server ในทุกกรณี
  • Respond only to know client computers = ยินยอมให้ Client เชื่อมต่อเข้ามาที่ WDS Server ได้ (เฉพาะเครื่อง Client ที่มี Computer Object อยู่ในระบบ AD DS เรียบร้อยแล้ว)
  • Respond to all client computers (know and unknown) = ยินยอมให้ทุก Client เชื่อมต่อเข้ามาที่ WDS Server ได้(มี Option คืออาจจะเลือกให้ผู้ดูแลระบบเป็นคน Approve ก่อนให้ Client เข้ามาเชื่อมต่อได้เป็นครั้ง ๆ ไปโดยผ่าน WDS Console)




4. หลังจากเลือกแนวทางในการเชื่อมต่อแล้ว Wizard จะดำเนินต่อไปจนเสร็จสิ้นซึ่งหลังจากนี้ตัว Wizard จะช่วย Start WDS Service ขึ้นมาเพื่อให้ใช้งานดังรูปด้านล่าง


จบขั้นตอนสำหรับการติดตั้งและปรับแต่งเพื่อนำไปใช้งานเบื้องต้น ข้อมูลเพิ่มเติมเกี่ยวกับ Windows Deployment Services สามารถค้นหาต่อได้ที่
http://go.microsoft.com/fwlink/?LinkId=88439
