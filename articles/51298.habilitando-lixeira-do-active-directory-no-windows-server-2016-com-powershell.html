Procedimento

Primeiramente acesse o modulo de Active-Diretory do AD no seu Servidor.

Com o modulo aberto digite o comando abaixo onde seudominio e o domínio do sem ambiente.

Enable-ADOptionalFeature -Identity “CN=Recycle Bin Feature,CN=Optional Features,CN=Directory Service,CN=Windows NT,CN=Services,CN=Configuration, DC=<seudomínio>,DC=<seudomínio>” -Scope ForestOrConfigurationSet -Target “<seudomíno. seudomíno >”

E para restaurar algum objeto, utilize o comando abaixo com o nome da conta deletada.

Comando PowerShell para restaurar o objeto:

Get-ADObject -ldapfilter:”(sAMAccountName=nome da conta deletada)” -IncludeDeletedObjects | Restore-ADObject

Procedimento simples e fácil de ser realizado.
