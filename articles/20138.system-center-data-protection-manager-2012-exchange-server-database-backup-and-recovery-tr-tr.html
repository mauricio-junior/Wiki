System Center Data Protection Manager 2012 ile Exchange Server Database'lerinin backup'larının alınması ve bu backup'lardan restore işlemini adım adım uygulayarak gerçekleştirebilirsiniz. Alınan backup'lardan, database bazında ve/veya user mailbox bazında restore işlemini sorunsuz olarak gerçekleştirilebilir.
