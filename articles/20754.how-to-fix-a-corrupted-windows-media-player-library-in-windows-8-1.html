To repair or fix a corrupted Windows Media Player Library go through the following steps:-

Run Windows Media Player Troubleshooter



2. Select View all option present in the left side.




3. From the list scroll down to Windows Media Player Library and click on it to launch Windows Media Player Troubleshooter wizard.





4. Click Next to proceed and recreate media library.


Rebuilding Windows Media Player Database

1. Press Win Key+R to launch Run menu.




2. Type %LOCALAPPDATA%\Microsoft in the box and press the Enter Key.

3. Locate Media Player Folder.





4. Right click on it and select Delete.

[Windows Media Player will automatically recreate the database]

5. Restart Windows Media Player.
