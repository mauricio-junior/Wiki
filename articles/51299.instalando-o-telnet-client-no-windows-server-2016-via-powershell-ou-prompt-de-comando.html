Procedimento Simples e facil de ser realizado.

Para instalar o cliente do telnet nos servidores Windows Server 2016 via prompt de comando ou Powershell siga os passos abaixo.

Abra o powershell como administrador e execute um dos comandos abaixo.

Com prompt de comando esse comando:

dism /online /Enable-Feature /FeatureName:TelnetClient

Com powershell esse comando.

Install-WindowsFeature -name Telnet-Client



Finalizando seu telnet cliente esta instalado e habilitado.
