Introdução

Nesse procedimento vamos mostrar como Criar um hotspot Wi-Fi no Windows 8.

Aplica-se


Windows 8

Configurando


Pressione Windows + R e Executar e digite ncpa.cpl e tecle enter.

 

Quando a pasta das suas conexoes de rede abrir ,clique com o botão direito do mouse no seu adptador de rede sem fio e  selecione Propriedades.


 
Mude para a aba sharing permitindo que outros dispositivos possam usar a sua conexão, selecionando a primeira opção e, em seguida desmarque a segunda opção e clique no botão OK.

 

Agora novamete clique em Windows + X para iniciar um prompt de comando modo Admin.

 


Agora vamos configurar a rede sem fio, utilizando o comando netsh.

No prompt digite netsh wlan set hostednetwork mode=allow ssid=” Wireless Network Connection 2” key=”12345.lhlc”



Onde SSID é o nome da sua rede e key é a senha que voce vai disponibilizar para os usuarios utilizarem, o ponto já sera criado como WPA2-PSK (AES).

Agora digite o comando:

netsh wlan start hostednetwork



Finalizando o comando Agora é so começar a acessar seu hotspot.

Existe tambem comando show, ele pode te mostrar o número de clientes conectados à sua rede:

netsh wlan show hostednetwork


Conclusão


O procedimento acima  mostra como transformar seu Windows 8 em um ponto de acesso Wifi.
