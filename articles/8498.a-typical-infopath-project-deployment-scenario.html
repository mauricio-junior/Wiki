A Typical InfoPath Project Deployment Scenario

Here is a typical scenario for InfoPath project deployment:
  • InfoPath forms will be published as content types for reuse purpose.
  • A form library will be created to host InfoPath form instances
  • SharePoint security groups will be created and applied to the form library
  • Users need to be added to the SharePoint security groups.
