Install Process: Server 8

Na het voltooien van uw installatie, log in:

In mijn voorbeeld gaan wij de naam SRV-APP-8 en als domain-naam gokmania.local gebruiken.

Installatie: SQL

SQL Server 2012 gaat testen uitvoeren. Bij mij gaf dit een waarschuwing omdat mijn FIREWALL open stond. Manueel moeten de poorten in de Firewall geopend worden voor een goede communicatie.

 

Selecteer Evaluation Version of geef uw Product Key in

 

 

Accepteer de “License Terms”

 

 Kies “SQL SERVER Feature Installation”

 

Selecteer alle features dat SharePoint gaat gebruiken. Een (1) groot voordel tegen over de vorige versie is dat men nu “Reporting Services Add-in for SharePoint Products” hebben. Men kan nu deze direct installeren en moetten we niet apart gaan instellen.

 

 Geeft een named instance of laat het de naam als defL SERVER Feature Installation”

 

Selecteer alle features dat SharePoinault.

 

 

 

 

 Alweer een grote verandering. De Account Namen worden door SQL Server gecreëerd. Een groot verandering tegenover de vorige versie. Nu kunnen we direct door de Account Name in de Event Viewer weten welke service er niet meer draait.

Selecteer Mixed Mode en geef een password voor SA account

Voeg uw locale Administrator als Farm administrator toe.

 

 Voeg uw locale Administrator als Farm administrator.


Installeer Reporting Services zonder het te configureren.

 

 En de installatie kan beginnen.
