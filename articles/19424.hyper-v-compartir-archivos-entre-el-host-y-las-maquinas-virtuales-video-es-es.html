Una de las grandes ventajas de Hyper-V es su integración en el sistema Windows 8 y esto hace que un gran número de usuarios lo utilicen para implementar multitud de sistemas para probarlos o simplemente para explorarlos o experimentar con ellos, pero una de las grandes piedras del camino ha sido sin duda la dificultad de compartir archivos entre el sistema Host y  los sistemas virtuales.

A continuación se muestra el procedimiento a llevar a cabo para poder compartir archivos entre las maquinas virtuales y los sistemas anfitriones que es muy sencillo ya que bastará con generar un recurso compartido en el host para después accederlo desde los sistemas virtualizados.

Veamos a continuación unas imágenes, en la primera de ellas vemos el momento en el que se agrega un usuario para acceder al recurso.



Esta imagen muestra el momento en el que accedemos al recurso compartido y nos pide credenciales para acceder.


Superados estos pasos podremos acceder al recurso y dependiendo de los permisos otorgados podremos acceder con privilegios de solo lectura, o de lectura y escritura.
