Nesse artigo vou falar sobre um erro que ocorre no Mcafee Policy Orchestrator. Quando instalamos o servidor de antivírus, sua integração com a serviços do Active Directory é efetuada, com isso, o banco SQL Express utilizado pela ferramenta, é autenticada no domínio. Geralmente esse erro ocorre quando as credenciais do domínio são alteradas em seu ambiente. 

Veja o erro apresentado na imagem abaixo:

 

Para solucionar esse problema, devemos reconfigurar as credenciais que fazem a conexão com o banco de dados. Para efetuar esse procedimento digite em seu navegador o seguinte endereço:

https://localhost:8443/core/config

Com esse endereço, podemos redefinir a senha administrativa e efetuar o teste com o banco de dados. Confira a configuração na imagem abaixo:
 


Após, precisamos reiniciar os serviços para que essa alteração funcione.

Importante: Se o serviço não for reiniciado, o procedimento não funcionará.

Pronto ! Sua console estará acessível e pronta para receber seu gerenciamento e sua tarefas.
