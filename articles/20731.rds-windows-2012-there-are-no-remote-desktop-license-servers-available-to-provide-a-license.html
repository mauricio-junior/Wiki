No Windows Server 2012 Remote Desktop, recebemos a seguinte mensagem de erro quando os 120 dias de licença do servidor terminam:
"there are no Remote Desktop License Servers available to provide a license."


Mesmo que após o termino dos 120 dias você tente adicionar licenças de Remote Desktop, o erro continua a aparece.

Para corrigir o problema, devemos executar os seguintes procedimentos:
A

cessar o Registro do Servidor (regedit.exe) e acessar a seguinte chave:

HKLM\System\CurrentControlSet\Control\Terminal Server\RCM

​Dentro desta chave "GracePeriod" que devemos excluir. Sempre executando um backup do registro antes de executar o procedimento.

Obs: ​É necessário ter permissão nesta chave para poder excluir ela.
