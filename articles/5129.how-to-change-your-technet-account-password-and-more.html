To change your password and other account properties,follow these simple instructions:
  1. Visit the Windows Live Account Overview page

  2. Find the Password: option and select Change.

  3. When the 'Change your password' page loads, enter your previous password in the Old password field.



  4. Type your desired password in the Type new password and Retype new password fields.

  5. (Optional) Select Make my password expire every 72 days

  6. Click Save

Congratulations!  Your password has now been changed.
