Issue

A virtual machine running Windows 8 is configured with less than the recommended amount of RAM, which is 1 GB.

Impact

The guest operating system and applications might not perform well. There might not be enough memory to run multiple applications at once. This impacts the following virtual machines: <list of virtual machines>

Resolution

Use Hyper-V Manager to increase the memory allocated to this virtual machine to at least 1 GB.
