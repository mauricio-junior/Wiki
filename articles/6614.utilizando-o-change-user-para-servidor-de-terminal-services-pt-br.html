Utilizando o Change User para Servidor de Terminal Services. (PT-BR)

Sempre trabalhamos com Terminal Services e as vezes nos deparamos com um problema relacionado a instalação e configuração de aplicativos, esses aplicativos queremos que seja acessível a todos os usuários sem que nos cause dor de cabeça ou que tenhamos que fazer alguma “gambiarra” no momento da instalação como:

  • Alterar registro
  • Alterar caminho da instalação
  • Alterar variável de ambientes.

Então segue uma dica muito importante quando for trabalhar com instalação de aplicativos em Terminal Services.

Essa dica vale para qualquer versão de S.O – Windows Server 2003 até Windows Server 2008 – R2

Instalar um programa usando o comando change user

  • Faça logon no servidor de terminal como administrador e feche todos os aplicativos.
  • Para abrir um prompt de comando, clique em Iniciar, aponte para Todos os programas, Acessórios e clique em Prompt de comando.
  • Digite change user /install e pressione ENTER.

             - Após digitar esse comando, o usuário entrará no modo de instalação:


  • Instale o programa em uma unidade local formatada do sistema de arquivos NTFS conforme orientado pelo programa de instalação.
  • No prompt de comando, digite change user /execute quando a instalação for concluída.

             - Após a instalação, digite o comando informado para retornar no modo de execução


  • Se aplicável, edite e execute o script de instalação do programa.

Observações

  • Para executar este procedimento, você deve ser membro do grupo Administradores no computador local ou deve ter recebido a autoridade adequada. Se o computador fizer parte de um domínio, é possível que os membros do grupo Administradores de domínio possam executar esse procedimento.
  • Se um programa solicitar a instalação de um programa adicional, você poderá deixar o programa inicial em execução.
  • Você pode digitar change user /query no Prompt de comando para ver se o modo de execução está habilitado e se o mapeamento do arquivo .ini está habilitado.
  • Você deve usar Adicionar ou remover programas para instalar programas sempre que possível.
