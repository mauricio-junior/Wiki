Sizlere Exchange Server üzerinde inceleyeceğimiz ve sıkla kullanacağımız kullanıcı tiplerinden bahsedeceğim. Exchange Server kullandığımız alanlarda maaliyeti düşürmek adına Kullanıcı planlanması gerekmektedir. CAL lisansı gerçeğini göz önünde bulundurmak gerekmektedir.

:#4d4d4d;background-color:#ffffff;"> CAL lisansından bahsedecek olursak, kullandığımız sistemin, client-user olan tarafından kullanım bedelidir.  Bu yüzden dikkatli yapılması gerekmektedir.

Tipler ;

  1. Mailbox User
  2. Mail Enable User
  3. Mail Contact

Mailbox User

Bilindiği gibi Exchange Server kullanıcılarını Active Diretoryden almaktadır.
Buna göre açağımız kullanıcı aynı zamanda Active Diretoryde olması gerekmektedir yada Açağımız kullanıcı aynı zamanda Active Diretoryde olacaktır.

Domanin Account  + Exchange User = Mailbox User olarak görebiliriz.


Buna göre Exchange User aynı zamanda Domain Account’a sahiptir.

Mail Enable User

Öyle zamanlarla karşılaşmaktayız ki Şirket içerisinde olan kullanıcı outsource olarak gönderilmiş olabilir . Kişini şirket içi kaynaklara erişebilmesi için Active Directory üyesi olup External Mail olarak tüm e-maillerin yönlendirilmesi durumudur.

Domain Account + External Mail = Mail Enable User olarak görebiliriz.

 

Mail Contact

Tamamen organizasyon dışında olan kişiler için geçerlidir. Tanımlanmış external e-mail adreslerini contract oluşturarak, organizasyonunuzda bir mail grubu oluşturup bir mail contact’ı oluşturabilme durumudur.
