Forefront Products

Forefront includes the following products:

  • Microsoft Forefront Identity Manager
Endpoint Security
  • Forefront Endpoint Protection
  • Microsoft Forefront Client Security
Server Security
  • Microsoft Forefront Protection 2010 for Exchange Server (previously Microsoft Forefront Security for Exchange Server and Sybari Antigen for Exchange)
  • Microsoft Forefront Security for Exchange
  • Microsoft Forefront Online Protection for Exchange
  • Microsoft Forefront Protection 2010 for SharePoint (previously Microsoft Forefront Security for SharePoint and Sybari Antigen for SharePoint)
  • Microsoft Forefront Security for Office Communications Server
  • Microsoft Forefront Server Security Management Console (FSSMC) (previously Antigen Enterprise Manager and Sybari Enterprise Manager)
  • Microsoft Forefront Protection Server Management Console (FPSMC) (the replacement of FSSMC for central management, Forefront Protection for Exchange 2010, and Forefront Protection for SharePoint 2010) More info: http://blogs.technet.com/b/fss/archive/2010/07/21/forefront-protection-server-management-update.aspx
  • Microsoft Antigen 9.0
Network Edge Security
  • Microsoft Forefront Unified Access Gateway (the successor to Intelligent Application Gateway (IAG) 2007)
  • Microsoft Forefront Threat Management Gateway (the successor to Internet Security and Acceleration Server)
