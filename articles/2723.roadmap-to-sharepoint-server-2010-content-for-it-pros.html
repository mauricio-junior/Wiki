Content resources

The SharePoint Server 2010 TechCenter (http://go.microsoft.com/fwlink/?LinkID=178512) is your primary gateway to in-depth technical information about SharePoint Server 2010. Through the TechCenter, you can access the following:
  • SharePoint Server 2010 Resource Centers (http://go.microsoft.com/fwlink/?LinkId=188958)   Resource centers provide specific links to more information specific to a particular feature or technology area of SharePoint Server 2010. You can find resource centers for features such as installation and deployment, Business Connectivity Services (BCS), enterprise content management, and enterprise search, to name a few.
  • SharePoint Server 2010 Downloads (http://go.microsoft.com/fwlink/?LinkId=188959)   Go to this page to download service packs, add-ins, and tools to help you optimize your SharePoint Server organization and trial software to help you get started.
  • SharePoint Server 2010 library   The library contains the most up-to-date IT pro content. This content is reviewed and approved by the SharePoint Server product team and will evolve over the life cycle of SharePoint Server 2010. Also make sure that you check out Newly published content for SharePoint Server 2010. This article contains a list of new or updated library content and is also available as an RSS feed (http://go.microsoft.com/fwlink/?LinkId=188960). It is updated whenever a new article is published or an existing article is updated.
  • SharePoint Server 2010 support (http://go.microsoft.com/fwlink/?LinkId=188961)   Locate support and troubleshooting resources for SharePoint Server. You can search the Microsoft Knowledge Base, TechNet, or blogs for additional help.

Offline resources

If you are working offline and need access to library articles, the following option is available to you:
  • SharePoint Server 2010 Technical Library in Compiled Help format (http://go.microsoft.com/fwlink/?LinkId=188962)   This downloadable .chm file is updated monthly and includes all articles published to the library as of the .chm file's release date.

Additional resources

The following visual resources supplement the library articles and can assist you when you plan your projects:
  • Video demos and training for SharePoint Server 2010   This article contains a list of SharePoint Server video demos intended for IT professionals.

  • Technical diagrams (SharePoint Server 2010)   These poster-size models provide a visual representation of recommended solutions and correspond to various articles in the library. You can modify these models by using Microsoft Office Visio 2007 to reflect your plans for SharePoint Server and how it will fit into your own environment.
