この手順は Craig Lussier によるものです。

複数のタグによるフィルタリングを行うための URL 規則は次の通りです。
http://social.technet.microsoft.com/wiki/tags/[tag1]/[tag2]/[tag3]/default.aspx
[tag1] などの場所を、URL エンコードされたタグの値で置き換えます。

例 1: SQL Server についての全てのポルトガル語記事を見たい

Wiki: 複数のタグでタグの結果をフィルタリングする方法 (ja-JP)

http://social.technet.microsoft.com/wiki/tags/pt_2D00_BR/SQL+Server/default.aspx
次のようにシンボルを変更することに注意してください。
  • "ダッシュ" は "_2D00_" に変更する必要があります。
  • "スペース" は "+" に変更する必要があります。
例 2: SharePoint についての全ての英語記事を見たいとします。それには、"en-US" と "SharePoint" のタグを使います。
http://social.technet.microsoft.com/wiki/tags/SharePoint/en_2D00_US/default.aspx
例 3: PowerPivot、Excel、そして既知の問題についての全記事を見たいとします。それには、"PowerPivot"、"Excel"、"Known Issues" のタグを使います。
http://social.technet.microsoft.com/wiki/tags/PowerPivot/Excel/known+issues/default.aspx
