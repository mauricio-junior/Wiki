Microsoft SharePoint Designer, Microsoft tarafından geliştirilen bir web tasarım yazılımıdır. Microsoft Office yazılımlarındandır fakat ayrı bir paket halinde satılmaktadır. Eski adı Microsoft FrontPage olan yazılımın amacı, web tasarımı yapmaktır. Adobe’un Dreamweaver yazılımına göre bazı özellikleri eksik kalmaktadır ancak Dreamweaver’dan sonra tercih edilen ikinci web tasarım yazılımıdır. Firmanın Expression Web Ürünü Web Programlama, SharePoint Designer ise tasarım yönünü yansıtmaktadır.
