Introdução

Na Microsoft Store você pode utilizar até 5 máquinas para instalar aplicativos, caso você precise remover algum dispositivos que você não vai mais utilizer e só seguir os procedimentos abaixo.

Removendo

Abra o ícone Windows Store no menu iniciar.
 
Depois Clique Win + C e selecione configurações. 


 

Em Configurações clique Sua conta.
 

Depois você vai todos os dispositivos que estão autorizados, agora clique no botão Remover no dispositivo que você não quer mais que fique disponível. 
 

 
Assim seu dispositivo será removido.

Conclusão

O Procedimento mostra como dispositivos autorizados no Windows Store.
