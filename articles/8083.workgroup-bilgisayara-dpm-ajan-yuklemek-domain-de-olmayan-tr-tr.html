  • Ajan yüklenecek bilgisayarda firewall kapatılır.
    (Kapatmak istemiyorsanız bu portlara izin vermelisiniz.)
  • DPM sunucu ile ajanın kurulduğu bilgisayar, birbirlerine Netbios adlarından ping atmalıdır.
    (Ortak DNS sunucusu kullanabilirsiniz, yada HOST dosyasına elle kayıt girebilirsiniz)
  • Ajan programı DMP server’da “C:\Program Files\Microsoft DPM\DPM\agents\RA\” bu path'de bulunur. Buradan kopyalayarak hedef bilgisayara kurulur. (Kurulum hokkabazı restart isterse mutlaka yapılmalıdır)
  • Kurulum bittikden sonra komut satırından bu komut çalıştırılır.
    (C:\Program Files\Microsoft Data Protection Manager\DPM\bin\SetDpmServer.exe -dpmservername dpm -IsNonDomainServer -UserName Dpmagentuser) enter’a bastığınızda açılacak user için bir parola sorar bilgisayarın parola politikasına göre bir parola girilir.
  • Komutun başarılı olarak tamamlandığı kontrol edilir.
  • DPM server tarafına geçilir. DPM konslundan "Management" sekmesi tıklanır. Install agent link’i tıklanır.
  • "Attach Agents" – "Computer in a workgroup….." seçeneği seçilir ne next butonuna basılır
  • Bilgisayarın netbios adı yazılır. User name alanına yukarıda oluşturulan user adı yazılır "dpmagentuser" parola da o user oluşturulurken girilen paroladır.
  • Next denir ve işlem tamamlanır.
