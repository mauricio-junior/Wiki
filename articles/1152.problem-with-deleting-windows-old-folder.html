Problem with Deleting Windows.old Folder

Trouble deleting Windows.old folder after upgrading from Windows Vista? Try the following steps!
 
 
Open Disk Cleanup

2. Select the C: drive and click on OK. 
3. Click on the Clean up system files button.
 
4. If prompted by UAC, then click on Yes.

5. Repeat step 2 above.

6. Check the Previous Windows installations box, and click on OK.
 
7. Click on Delete Files.
 
8. When Disk Cleanup finishes, the C:\Windows.old folder will be deleted.
