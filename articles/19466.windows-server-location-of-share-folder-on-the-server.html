Applies to: Windows Server 2003, 2008, 2008 R2, 2012 and 2012 R2

Context: There was a question on the forum wherein OP wanted to know about the actual folder path of a hidden share. To know more about the actual context, please refer the forum thread 
Where is the profile$ folder?

Info: Share$ is a hidden share, to hide any share from users, add a $ to the end of the share name e.g. Data$

Details: 

If you know the Shared folder path (which contains a hidden share) however, you would like to know the actual folder path of the hidden share on the server, you can locate the actual folder path of a hidden share by using steps mentioned below.

On the server in question, go to Computer Management (Run -> compmgmt.msc {Press Enter})

Shared Folders -> Shares

Look for Folder Path, see the snip below
