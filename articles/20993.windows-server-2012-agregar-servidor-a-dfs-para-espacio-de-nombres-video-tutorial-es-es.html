El servicio de DFS de Windows Server 2012 nos permite gestionar de una forma más eficaz carpetas compartidas de la red de forma que podamos compartir estas en una red dispersa geográficamente y de una forma totalmente transparente para los usuarios con las ventajas que ello conlleva.

En esta ocasión se va a explicar como proceder para unir un segundo servidor de DFS a la red y luego agregar a este el espacio de nombres creado en el primero, el segundo servidor consta de:

Sistema operativo Windows Server 2012
Servidor miembro del dominio. No es Controlador de Dominio

Teniendo esto solamente necesitaremos agregar el rol de DFS y una vez instalado agregar el espacio de nombres, a continuación puedes ver el proceso en vídeo.
