Xoş gördük. Exchange Server haqqındakı əvvəlki məqaləmizdə bəzi məqamlara toxunduğumuz üçün, bu gün ki, məqaləmizdə Exchange Server 2010-nun qurulması haqqında sizləri məlumatlandırmağa çalışacağam. Aşağıdakı keçiddən Exchange Server və lazım olan minimum tələblər haqqında informasiya əldə edə bilərsiniz.

- Exchange Server Haqqında

Bu tələblərələ tanış olduqdan sonra keçid alırıq Exchange Server 2010-nun sazlanması mərhələsinə. İlk məqaləmizdə qeyd etdiyimz keçiddən istifadə edərək Exchange Serverin 120 günlük Trial versiyasını yükləyərək Exchange-in bütün imkanlarından yararlana bilərsiniz. Exchange Serverin sazlanması üçün müəyyən komponentlərə ehtiyac vardır ki, biz bunların əsas hissəsini powershell üzərindən yükləyərək, əməliyyatlarımzı bir az sadələşdirmiş olacağıq.

a) İlk öncə Windows Server 2008 R2 üzərində Start -Run - PowerShell pəncərəsini açdıqdan sonra aşağıdakı əmrləri qeyd edərək ehtiyac olan komponentlərin yüklənməsini tamamlayaq. Əmrlər web səhifə üzərindən kopyalandğı zaman, bəzi hallarda sintaksis xətası verdiyi üçün, ehtiyac olacaq əmrləri mətn şəklində əlavə etdiyimz keçiddən əldə edə bilərsiniz. Həmin əmrləri kopyalayıb PowerShell pəncərəsinə yapıştırmağınız yetərli olacaq.

b) PowerShell üzərində birlikdə çalışdırılacaq əmrlər (sağ düymə vasitəsilə yaddaşa yazın)

Qeyd: Əlavə metod olaraq Exchange diski üzərində scripts bölməsinə daxil olaraq aşağdakı əmri icra edə bilərsiniz.

PS E:\Exchange Setup\scripts> ServerManagerCmd.exe -ip .\exchange-typical.xml

Aşağıdakı ehtiyac olan paketi server üzərinə qurduqdan sonra artıq ilkin mərhlə üçün ehityac olan addımları tamamlamış olacağıq.

c) Microsoft Office 2010 Filter Packs

1. Yuxarıdakı addımları tamamladığımız üçün Exchange qovluğundan və yaxud daxil etdyiniz disk üzərindən Setup deyərək açılacaq pəncərədən İnstall Microsoft Exchange bölməsinə daxil oluruq.


2. Exchange haqqında qısa məlumat aldıqdan sonra Next deyərək yolumuza davam edirik.

3. İ accept.

4. Exchange Serverin təkmilləşməsi, analizlərin daha yaxşı aparılması və s. kimi məsələlərdə Microsofo şirkətinə dəstək olmaq istəyirsinizə Yes əks təqdirdə No deyərək davam edirik.

5. Exchange Serverin rolları haqqında başqa bir məqalədə geniş izahat verəcəyimiz üçün Typical Exchange Server İnstallation bölməsini seçərək davam edirik. Automatically install Windows Server roles and features required for Exchange - SP1 paketi ilə gələn bir yenilkdir, ehtiyac olan komponentlərin avtomatik yüklənməsinə imkan yaradır.

6. Exchange Organization - uyğun bir ad daxil edərək davam edirik. Apply Acite Directory split permissions securty model to the Exchange organization - bu yenilik vasitəsilə Active Directory və Exchange Server idaretməsini ayrı-ayrı istifadəçi və qruplara ayırmaq mümkündür.

7. Əgər strukturunuzda Outlook 2003 mövcud deyilsə bu bölmədə No deyərk davam edirik.

8. İnternet üzərindən daxil olmaq üçün Client Access Serverə aid bir adres daxil edirik.

9. CEİP - no deyərək davam edirik

10. İnstall. Bu bölmədə çıxan bildiriş /PrepareAD əmrini öncə icra etmədiyimizlə bağılıdır. Bu addımı Exchange özü gerçəkləşdircək.

11. Finish. View Setup Log - bölməsindən gerçəkləşmiş əməliyyatlar haqqında geniş məlumat əldə edə bilərsiniz.

12. Reboot.

11. Finish. View Setup Log - bölməsindən gerçəkləşmiş əməliyyatlar haqqında geniş məlumat əldə edə bilərsiniz.

12. Reboot.
