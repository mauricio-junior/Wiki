Aplikasi Office 2013 terintegrasi secara penuh dengan layanan penyimpanan online dari Microsoft yaitu SkyDrive, sehingga jika Anda hendak menyimpan file atau dokumen yang Anda buat maka lokasi penyimpanan default yang dipilih oleh Office 2013 adalah SkyDrive tersebut.

Jika memang semua data-data Anda tersimpan di SkyDrive, hal ini jelas sangat membantu, namun kebanyakan para pengguna Office tampaknya masih memilih untuk menyimpan data di hardisk lokal sehingga mungkin perlu juga untuk merubah lokasi penyimpanan default ini. Caranya sebagai berikut.

Buka aplikasi yang hendak Anda rubah lokasi penyimpanan defaultnya, misalkan saja aplikasi Word 2013. Kemudian klik menu File dan klik Options hingga jendela Word Options terbuka.

Klik pada pilihan Save di sebelah kiri kemudian disebelah kanan beri tanda centang pada opsi Save to Computer by default. Jika perlu, Anda juga dapat menentukan dimana lokasi default penyimpanan tersebut di hardisk Anda, yaitu dengan mengklik tombol Browse pada opsi Default local file location.

Berikutnya klik tombol OK untuk menutup jendela WOrd Options tersebut. Dan sekarang setiap kali Anda hendak menyimpan dokumen, maka secara default lokasi penyimpanan akan diarahkan ke hardisk lokal dengan lokasi yang telah Anda tentukan sendiri.

Lokasi penyimpanan default ini berlaku untuk aplikasi Word 2013, Excel 2013, dan PowerPoint 2013.
