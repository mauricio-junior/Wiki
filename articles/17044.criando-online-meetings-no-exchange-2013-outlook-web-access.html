Visão Geral


Uma das novas funcionalidades no Exchange Server 2013 CU1 é a habilidade de trabalhar com Online Meetings usando o Outlook Web Apps. 


Pré-Requisitos

 
Para ativar a feature configure as seguintes funções:
  • O usuário agendando a reunião deve estar habilitado em um Lync 2013 pool;
  • O usuário agendando a reunião deve estar hospedado no Exchange 2013 CU1;
  • Configurar a autenticação Server-to-Server (S2SOAuth) Entre Exchange 2013 e Lync 2013, documentado no artigo Configuração do Server-to-Server authentication (S2SOAuth) Entre Exchange 2013 e Lync 2013; 
  • O serviço Lync Autodiscover configurado. Adicione os registros lyncdiscover.<domínio SIP> e lyncdiscoverinternal.<domínio SIP> nos DNS interno resolvendo para o nome do Front End;

Agendando Reuniões


Para criar uma reunião online logue no OWA e acesse o calendário


Clique em New Event


Se o usuário se cumprir em todos os pré-requisitos o novo botão Online Meeting será exibido.


Clicando no botão as informações da sala são inseridas 


Com todos os campos preenchidos clique em Send para enviar o convite para os usuários


Ingressando na Reunião


Para ingressar na reunião acesse o agendamento no calendário e nas propriedades da sala clique no botão Join
