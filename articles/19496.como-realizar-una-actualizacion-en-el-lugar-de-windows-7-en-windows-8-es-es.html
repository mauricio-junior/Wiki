¿Ya tienes una máquina de Windows 7 y ahora comprado Windows 8? ¿Planea hacer un upgrade en el lugar, así que no tienes que volver a instalar todos los programas desde cero? Entonces este es el artículo adecuado para usted. Se le mostrará lo fácil se realiza la actualización en el lugar para actualizar Windows 7 a Windows 8.

>> Siempre copia de seguridad de sus datos, por lo que si pase lo que pase durante la actualización, sus datos estaría a salvo.

 

  1. Ok, así que ponte tu Windows 8 DVD disco en la unidad de DVD y luego el AutoPlay debe popup al elegir una acción, haga clic en Ejecutar setup.exe




    Si la función de reproducción automática no funciona en el paso anterior, puede abrir el ordenador y luego haga doble clic en la unidad de DVD



  2. Si usted tiene User Account Control (UAC) habilitada, luego pedirá que si desea permitir el programa para hacer cambios a este equipo, haga clic enSí



  3. Se prepararán el asistente de instalación de Windows 8



  4. El asistente de instalación se iniciará y le preguntará si desea obtener las últimas actualizaciones, se recomienda ir en línea si su máquina está conectada a internet. Haga clic en siguiente para continuar



    El sistema comprobará si las actualizaciones necesarias y los descargará



  5. Entonces se le pedirá que escriba la clave de producto para la licencia de Windows 8. Una vez escrito, haga clic en siguiente



  6. En la página de términos de la licencia, acepta los términos y haga clic en Próxima



  7. Su tiempo que usted elige lo que quieres hacer con tus datos y la configuración existente. ¿Quieres conservar ambos? ¿sólo archivos personales? ¿o no?

    Configuración de Windows:Se mantendrá la configuración de Windows como facilidad de acceso, tu fondo de escritorio o favoritos de Internet e historia.
    Archivos personales:Nada guardado en la carpeta de usuario se considera un archivo personal, como los documentos y carpetas de escritorio.
    Aplicaciones:Para las aplicaciones que son compatibles con Windows 8, algunas aplicaciones no son compatibles con Windows 8 y tendrás que reinstalarlos.
    Nada:Borrar todo y reemplazar la versión actual con una copia de Windows 8. Sus archivos personales se moverán a una carpeta windows.old

    Va estar haciendo una actualización del sistema operativo y quisiera conservar todo lo que es y guardar mis datos, así que voy a elegir la primera opción, haga clic en siguiente para continuar

  8. Instalación de Windows 8 comprobará si todo está listo y si cualquier incompatible aplicaciones instalación en tu Windows 7. Si no hay aplicaciones incompatibles se encontraron, entonces salta al paso # 9, más seguir leyendo.



    Si hay una aplicación que no es compatible con Windows 8, luego configuración le informará de ello y tendrá que desinstalar para continuar.

    Por ejemplo, en la siguiente instantánea, puede ver esa instalación ha detectado que Microsoft Security Essentials no es compatible con Windows 8 y se debe desinstalar. Haga clic en desinstalar



    Haga clic en Actualización

    Si las aplicaciones incompatibles no más encontrada, se le pedirá que reinicie su equipo. Haga clic en reiniciar



    Reiniciará su Windows 7 y una vez que se registro en la parte posterior, la instalación de Windows 8 se reanudará, preguntando si desea continuar con la configuración que usted ha seleccionado previamente o quieres empezar de nuevo. Elija continuar desde donde lo dejé y haga clic en siguiente



  9. Instalación de Windows 8 está ahora listo para instalar Windows 8 con la configuración que ha elegido. Haga clic en instalar para comenzar



  10. Comenzará a instalar Windows 8





    El PC se reiniciará varias veces.

            

  11. Entonces la voluntad de Personalizar ventana que aparece, elija un color te gusta para tu ventana 8, elegiré Orange y haga clic en Próxima



  12. En la ventana de configuración , elija usar una configuración expresa



  13. En la ventana de iniciar sesión en tu PC , escribe tu cuenta de Microsoft y haga clic en siguiente



    Escriba la contraseña de tu correo electrónico y haga clic en siguiente



    Usted necesitará agregar información de seguridad para tu cuenta de Microsoft, tales como número de teléfono y correo electrónico alternativo.





  14. Después de crea tu cuenta, configuración de Windows 8 comenzará a construir su máquina y aplicar la configuración que ha elegido previamente utilizando al asistente.

            

  15. Una vez hecho esto, se mostrará el nuevo menú Inicio de Windows 8, y con esto, tu Windows se actualiza correctamente. Felicidades.



Resumen

Microsoft ha hecho un asistente de configuración fácil y nueva para instalar o actualizar Windows 8.
