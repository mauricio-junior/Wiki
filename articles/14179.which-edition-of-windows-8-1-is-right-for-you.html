There are four editions of Windows 8.1 available and you could get any of these editions depend on your needs.

1) Windows 8.1: You could buy this edition from retail store or online shop and it contains basic features of Windows 8.1 If you are planning to use Windows 8.1 for personal use or family use, it is the recommended edition.

2) Windows 8.1 Pro: You could buy this edition from retail store or online shop and it contains all features of Windows 8.1 and some additional features for driver encryption and connecting to domain. If you plan to get a PC for professional work or for a company which you need to join domain or do some encryption, Windows 8.1 Pro is the best.

It is edition which comes pre-install in some tablet PCs and you have a Microsoft Office pre-install in this edition. If you plan to get a tablet and you are on go and moving around, it is the edition that you will need.

4) Windows 8.1 Enterprise: It contains all features of Windows 8.1 and Windows 8.1 Pro and some additional features that are for IT Administration and Management in a company such as Windows To Go.  If you are an IT Professional or you want to get PC for large organization, this is the edition that you will need.

Notes:

When you buy a new PC, it could have Windows 8.1, Windows 8.1 Pro or Windows 8.1 RT pre-install on it. Therefore when you buy a new PC, look for its edition.

Windows 8.1 Enterprise is available through Microsoft subscription and licensing and is not available in market.

Windows Media Center is not part of Windows 8.1 editions; however you could install it as a feature in Windows 8.1 Pro only.

If you are using any edition of Windows 8 except for Windows 8 Enterprise, you could upgrade to Windows 8.1 for free. Run Windows Update and install all updates and check Windows Store to install Windows 8.1 . For those who are using Windows 8 Enterprise,  your IT administrator should get license through Microsoft Licensing portal and deploy it for you.
