Once a Virtual Machine Manager (VMM) for System Center 2012 Service Pack 1 (SP1) add-in has been packaged, it can be installed and uninstalled through the console. For more information about managing add-ins, see Managing Add-ins.

To install an add-in package

  1. In the console, click on the Settings workspace.

  2. Click on the Console Add-Ins setting.

  3. On the ribbon, click on the Import Console Add-In button.

  4. Follow the wizard to import your add-in .zip file.

  5. The add-in will immediately be ready for use.

To uninstall an add-in package

  1. In the console, click on the Settings workspace.

  2. Click on the Console Add-Ins setting.

  3. In the list of add-ins, select a single add-in item.

  4. On the ribbon, click on the Remove button.

  5. You will be prompted with a confirmation window.

  6. If you click the Yes button in the confirmation window, the add-in will be immediately removed from the console.
