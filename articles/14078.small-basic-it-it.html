Small Basic è un linguaggio per principianti basato .NET sviluppato da Microsoft. E' stato in fase di sviluppo per diverso tempo ed è stato rilasciato recentemente poco più di un anno fa. E' fortemente localizzato ed è disponibile in più di 20 lingue. Small Basic è attualmente una versione ridotta di Visual Basic.NET, eseguito su .NET Framework 3.5.

Estensioni

Se si ha bisogno di utilizzare in un programma una funzione che non è disponibile nella libreria principale di Small Basic's, è possibile sviluppare un'estensione in qualsiasi altro linguaggio basato su .NET (http://social.technet.microsoft.com/wiki/contents/articles/13340.how-to-create-a-small-basic-extension.aspx).  Se non si vuole sviluppare un'estensione per conto proprio, è possibile scaricarne alcune già disponibili, come PowerExtension (https://powerextension.codeplex.com/) o LitDev's Extension (http://litdev.hostoi.com/).

Community

Esiste una community molto attiva di Small Basic nei Forums reperibili all'indirizzo http://social.msdn.microsoft.com/Forums/en-US/smallbasic/threads. 

Iniziare

Se si vuole impararare a programmare, Small Basic è un ottimo modo per iniziare! Può essere scaricato dall'indirizzo http://smallbasic.com/.
