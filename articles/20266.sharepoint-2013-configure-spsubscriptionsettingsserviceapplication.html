The service application "SPSubscriptionSettingsServiceApplication" is not visible from Central administration, this service application needs to be configured with PowerShell script. Below is the script which can be used for configuring this service application:

$serviceAccount = Get-SPManagedAccount "domain\accountName"
 
$appPool = New-SPServiceApplicationPool  -Name SettingsServiceAppPool -Account $serviceAccount
 
$serviceApp = New-SPSubscriptionSettingsServiceApplication  –ApplicationPool $appPool –Name SettingsServiceApp  –DatabaseName <SettingsServiceDB>
 
$serviceAppProxy = New-SPSubscriptionSettingsServiceApplicationProxy  –ServiceApplication $serviceApp
