En Windows Server 2008 R2 ya aparece el concepto de Papelera de Reciclare, pero esta Papelera es bastante tediosa de usar, es decir: para habilitarla tenemos que hacerlo con PowerShell y la recuperación de los objetos no es una recuperación trivial, ya que se tiene que acceder a verificar donde esta el objeto, que objetos hay, usar PowerShell para recuperarlos... Es cierto que funcionaba muy bien, pero como os he dicho, su uso era bastante tedioso…

Ahora en Windows Server 2012, se nos ha incluido una interfaz grafica para gestionar esta Papelera de Reciclaje, de este modo conseguimos mejorar la respuesta por parte de los administradores en la recuperación de objetos borrados en el Directorio Activo. Y nos ofrece una Vista consistente para la detección de objetos eliminados.

Que requisitos para poder usar la Papelera necesitamos:

  • Nivel funcional de bosque, ha de ser: Windows Serve 2008 R2 o superior.
  • Activar la característica de la Papelera de reciclaje.
  • Y desde un equipo que tenga la consola de Active Directory Administrative Center de Windows Server 2012, podríamos hacer todo el proceso de recuperación de objetos del Directorio Activo.
  • Los objetos a poder recuperar, una vez que han sido eliminados del AD, tienen por defecto un tiempo de vida de 180 día en la Papelera.
