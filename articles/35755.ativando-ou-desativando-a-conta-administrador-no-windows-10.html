Resumo

A conta de Administrador do Windows 10 oferece acesso e controle total sobre o sistema e utilizada para instalar aplicativos entre outros processos que seja necessário para a conta.


Detalhes

Como habilitar: Primeiramente acesse o Prompt de Comando do Windows 10.

Agora com o prompt de comando aberto digite o comando abaixo:

net user administrador /active:yes

E para Desativar Execute o comando abaixo:

net user administrador /active:no

Finalizando o seu usuário administrador vai está habilitado ou Desabilitado.
