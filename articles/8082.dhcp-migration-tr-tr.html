DHCP Migration (tr-TR)

Eski serverınızda bulunan DHCP servisinin yeni sunucunuza aktarmak için aşağıdaki adımları uygulamanız yeterli.

  • CMD açılır.
  • netsh dhcp server export c:\dhcpyedek.txt all
    • utun çıktısında bir Scope görülürse konsoldan yada aşağıdaki komut ile o scope silinir. (ip address yazan yere sizin scope'unuzda görünen ip yazılır)
  • netsh dhcp server delete scope IPaddress dhcpfullforce
  • Emin olunduktan sonra aşağıdaki komut ile yedek dönülür.
  • netsh dhcp server import c:\dhcpyedek.txt all
  • Komutun başarılı olarak tamamlandığı gözlenir.
  • Yeni server’dan DHCP konsol açılır ve her şeyin düzgün olup olmadığı kontrol edilir.
  • Eğer DHCP server adının yanında yeşil yukarı ok yoksa DHCP sunucusu Authorize ve Activate edilmelidir.
