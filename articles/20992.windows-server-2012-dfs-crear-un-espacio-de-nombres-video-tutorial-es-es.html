Windows Server 2012 con la ayuda del servicios de DFS nos permite mantener sincronizados los archivos de una red en diferentes ubicaciones de una forma muy sencilla, para ello hace uso de unos "Espacios de nombres" que nos permiten crear una carpeta de red que dentro contiene otras carpetas que pueden encontrarse en ubicaciones geográficas distintas, de esta forma cuando el usuario accede a una carpeta puede estar viendo información de diferentes ubicaciones sin saberlo y de forma totalmente transparente lo que facilita mucho la interacción entre distintas ubicaciones.

En el siguiente vídeo verás como crear un espacio de nombres pero antes encontrarás un enlace con mucha más información sobre DFS y los espacios de nombres.

Introducción a DFS - Presentación
