Merhabalar;

Bir kaç gündür müşterilerimizde terminal server kullanımında "Because a security error, the client could not connect to the remote computer. Verify that you are logged on to the network, and then try connecting again" hatasının alındığını gördüm.

Sorun tam olarak şöyleydi;

Kullanıcılar el terminallerini kullanarak Server 2008 R2 Terminal Server'a bağlanmaya çalıştıklarında bu hatayı alıyorlar ancak laptop veya PC'ler üzerinden bağlantı yaptıklarında bir sorun ile karşılaşmıyorlardı.

Bu sorunun çözümü için Terminal Server lisans sunucusunu Telefon ile yeniden etkinleştirmemiz lazım ancak onun öncesinde aşağıdaki işlemleri yapmalıyız.

Aşağıdaki işlemleri Registry üzerinde yapacağımız için registry'nin yedeğini almamız olası sorunlarda geri dönüş imkanı sağlayacaktır.

1)Regedit'i açalım ve HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Terminal Server\RCM değerine gelelim.

2)Aşağıdaki değerleri silelim ve registry editorunden çıkalım.

Certificate
X509 Certificate
X509 Certificate ID
X509 Certificate2

3)Remote Desktop Session Host Configuration'ı açalım ve Licensing kısmından Remote Desktop Licensing Mode'u çift tıklayalım.

4)Specified License Servers altından sunucumuzu seçelim ve Remove butonuna basalım.

5)Sunucumuzu yeniden başlatalım.

6)Sunucumuz açıldıktan sonra Remote Desktop Licensing Manager'ı açalım.

7)Sunucu ismimiz üzerinde sağ tıklayalım ve Advanced kısmından Deactivate Server komutunu verelim.

Gelen ekranları Next next diyerek tamamlayalım ve Finish ile Deactive işlemini tamamlayalım.

8)Sunucumuz üzerinde sağ tıklayalım ve Active Server komutunu verelim.

9)Gelen Ekranı Next ile geçelim.

10)Connection Method olarak Telephone'u seçelim.

11)Gelen ekranda Turkey'i seçelim.

12)Gelen ekranda Lisans Belgelerimizi hazırlayalım ve 0800 211 39 39 numaralı Microsoft Etkinleştirme hattını arayalım.

13)Sonrasında sizden istenen bilgiler doğrultusunda verilen Key'leri belirtilen kutuculara girelim.

14)Son olarak Remote Desktop Session Host Configuration'ı açalım ve Licensing kısmından Remote Desktop Licensing Mode'u çift tıklayalım.

15)Specified License Servers altından sunucumuzu seçelim ve Add butonuna basalım

16)Known License Servers içinden Lisans sunucumuzu seçelim ve Add butonuna basarak sağ tarafa ekleyelim.OK ve OK diyerek işlemi tamamlayalım.
