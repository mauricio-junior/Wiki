AlignWithPipelineExtensibilityXML

Avoid one of the most common pitfalls with Content Enrichment Web Service (CEWS) in SharePoint 2010. Maintain an XML file describing the contract between SharePoint and your CEWS service. essor at the end of your processing pipeline.

Configuration

<documentProcessor dllFile="StandardDocumentProcessors.dll"   className="StandardDocumentProcessors.AlignWithPipelineExtensibilityXML">
   <parameter name="LogFile">..\logs\AlignWithPipelineExtensibilityXML.log</parameter>
   <parameter name="PipelineExtensibilityXML">C:\FASTSearch\etc\PipelineExtensibility.xml</parameter>
</documentProcessor>

Configuration Options

The only important option is a path to the PipelineExtensibility.xml file.

LogFile

Optional log file.

PipelineExtensibilityXML

Absolute or relative path to your PipelineExtensibility.xml file.
