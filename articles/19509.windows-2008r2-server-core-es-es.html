Windows 2008/R2 Server Core

Server core es un sistema operativo que está teniendo los componentes mínimos de un server 2008/R2 y utilizar un menos hardware requisitos y menos tareas de mantenimiento.
Casi su cmd base y allí son sólo algunas opciones vienen como una gui (Ex hora y fecha).

Funciones de servidor disponibles en Server Core

  • Active Directory (AD)
  • Servicios de directorio ligero de Active Dirdiv class="r2 fiji-r2">
Servidor DHCP
  • Servidor DNS
  • Servicios de archivos
  • Hyper-V
  • Servicios de impresión
  • Servicios de streaming de medios de comunicación
  • Servidor Web (IIS)
  • Servicios de certificados de Active Directory

     

     

    Ventajas de Server Core

    • uso mínimo de hardware
    • Menos tareas de mantenimiento
    • Reducir la interfaz de ataque
    • Requiere menos espacio en disco (4-5 GB)

     

    Configuración inicial del servidor central con CMD/PowerShell

     

    Para hacer la configuración inicial (cambiar nombre de host, dirección ip, cambiar el nombre de red, activar) podemos utilizar el cmd y PowerShell. Pero windows 2008 R2 allí es una herramienta llamada sconfig.cmd que está dando opciones más fáciles para la misma configuración.

    Cómo

    Cambiar nombre de equipo

     

    Obtener el nombre del equipo nombre tipo hostname y pulse enter

     

    Cambiar el nombre de computadora tipo netdom renamecomputer WIN-1MKA1D339E9 /newname EAGLE

     

    WIN-1MKA1D339E9 ES UN NOMBRE ANTERIOR Y ÁGUILA ES UN NUEVO NOMBRE DE EQUIPO

    Prensa y y pulse enter

     

    Para reiniciar el ordenador tipo apagado – r – t 00

     

     

    Configuración de IP

    Tipo netsh(Network shell) y pulse enter

     

    Tipo int ipv4 y pulse enter

     

    Tipo agregar dirección "conexión de área local" 192.168.1.100 255.255.255.0 ("conexión de área local" es el nombre de una conexión de área local. Esto puede ser el hallazgo de escribir ipconfig/todo comando)

     

    Para agregar el tipo de puerta de enlace Agregar gateway "conexión de área local" dirección = 192.168.1.1 gwmetric-1

     

    Para agregar dnsserver tipo añadir dnsserver "conexión de área local" 8.8.8.8(8.8.8.8 es un servidor de dns de google puede agregar su propio servidor)
