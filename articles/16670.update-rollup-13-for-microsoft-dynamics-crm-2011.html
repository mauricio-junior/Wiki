    Update Rollup 13 for on-premises CRM servers was released on March 27th, 2013. It will be made available through Windows Update during April 2013. The build number of UR13 is 5.0.9690.3432.

    The release schedule of Update Rollup 13 for CRM Online has not been communicated. CRM Online has already received updates after the initial Update Rollup 12, which has increased the build numbers in Online organizations to 5.0.9690.3339 from the previous 5.0.9690.3233.

    Highlights


    Unlike in the previous Update Rollup 12 that delivered the cross-browser support for Microsoft Dynamics CRM, there are no major new functionalities included in UR13. The following enhancements have been made to the Microsoft Dynamics CRM Compatibility List:
    • Support for Windows Server 2012
    • Support for Active Directory Federation Services 2.1 (ADFS 2.1)
    Although UR13 technically supports Windows Server 2012, the CRM server installation files available on Microsoft Download have not yet been updated to allow using CRM 2011 on WS2012. The Self-Healing Setup (SHS) is needed for the installation on Windows Server 2012 and can be obtained by following the steps in the KB article "How to obtain the setup updates for Microsoft Dynamics CRM 2011" (for more details see this blog post). To upgrade the OS of an existing Windows Server 2008 with CRM 2011 installed to Windows Server 2012, please follow these upgrade instructions. For IFD environments, see the KB article "Configuring AD FS 2.1 with Microsoft Dynamics CRM 2011".
