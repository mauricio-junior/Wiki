Upgrade Data Quality Server

To upgrade the Data Quality Server component in DQS:

  1. Run the SQL Server 2012 RC0 setup, and upgrade your SQL Server Code-Named “Denali” CTP3 installation to the latest release. For instructions about upgrading SQL Server, see Upgrade to SQL Server 2012 using the Installation Wizard (Setup).
  2. Manually remove the .mdf and .ldf files for the two DQS databases: DQS_MAIN and DQS_PROJECTS. These files will be typically available at the following location on your computer: C:\Program Files\Microsoft SQL Server\MSSQL11.<InstanceName>\MSSQL\DATA.
  3. Run the command prompt.
  4. At the command prompt, change your directory to the location where DQSInstaller.exe is available. The DQSInstaller.exe file will be typically available at C:\Program Files\Microsoft SQL Server\MSSQL11.<InstanceName>\MSSQL\Binn. To change your directory:

    cd C:\Program Files\Microsoft SQL Server\MSSQL11.<InstanceName>\MSSQL\Binn
  5. Run the following command at the command prompt:

    dqsinstaller.exe –instance <InstanceName>

    NOTE: You must run the dqsinstaller.exe file with the –instance parameter. Otherwise, an error will occur.
