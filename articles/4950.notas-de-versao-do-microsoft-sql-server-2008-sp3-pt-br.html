1.0 Antes de instalar o Service Pack 3

Para obter uma lista dos bugs corrigidos no SQL Server 2008 Service Pack 3, consulte o artigo da Base de Dados de Conhecimento Microsoft 2546945.

2.1 Requisitos recomendados de espaço em disco para download, extração e instalação do SQL Server 2008 Service Pack 3.

Os requisitos de espaço em disco recomendados para download, extração e instalação do SP3 são aproximadamente 2,5 vezes maior que o tamanho do pacote. Os tamanhos dos pacotes do SP3 estão relacionados abaixo:

Pacotes do SQL Server Express (download):

  • SQLEXPR32_x86 (pode ser instalado em sistemas operacionais de 32 e 64 bits): 110 MB
  • SQLEXPR_x86 (pode ser instalado apenas em sistemas operacionais de 32 bits): 96 MB
  • SQLEXPR_x64: 123 MB

Todas as outras edições do SQL Server:

  • X86: 300 MB
  • X64: 367 MB
  • IA64: 359 MB

2.2 Os recursos do SQL Server 2008 R2 são exibidos no Assistente de Instalação do SQL Server quando você executa a atualização do SQL Server 2008 Service Pack 3

Você pode ver os recursos do SQL Server 2008 R2 incluídos na lista de recursos a serem atualizados pelo SQL Server 2008 Service Pack, na página Selecionar Recursos ou Pronto para Atualizar. Os componentes do SQL Server 2008 R2 não são afetados pela atualização do SQL Server 2008 Service Pack. Nenhuma ação é necessária.

Você pode ver os componentes do SQL Server 2008 R2 na lista de recursos, na página Selecionar Recursos ou Pronto para Desinstalar quando remover a atualização do SQL Server 2008 Service Pack. Os recursos do SQL Server 2008 R2 não são desinstalados durante esse processo. Nenhuma ação é necessária.

2.3 Todos os recursos de uma instância do SQL Server devem ter a mesma versão antes de aplicar uma atualização cumulativa

A atualização não funcionará, se você aplicar uma atualização cumulativa do SQL Server a uma instância do SQL Server na qual os recursos não estejam no mesmo nível de versão. Para evitar que isso ocorra, verifique se todos os recursos de uma instância do SQL Server estão na mesma versão de SQL Server.
