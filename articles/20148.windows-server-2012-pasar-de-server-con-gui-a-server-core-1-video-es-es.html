Una de las virtudes que nos ofrece Windows Server 2012 es la posibilidad de pasar de Server con GUI a Server Core sin la necesidad de reinstalar el servidor o migrarlo a otro cosa que en Windows Server 2008 no era posible.

En esta ocasión veremos como podemos deshacernos del entorno con GUI completo y dejar solamente el entorno Server Core que da mucho más rendimiento pero que nos obligará a lidiar con la interfaz de comandos del sistema por lo que es más que conveniente que hagamos algunas prácticas antes en un entorno con GUI para familiarizarnos con las tareas que vayamos a necesitar como pueden ser crear usuarios, recursos compartidos, establecer permisos, configurar dispositivos de red o instalar roles y características.
