Title
SQL Server Transactional Replication Latency Monitoring Tool

Description
The SQL Server Transactional Replication Latency Monitoring Tool improves the management of transactional replication environments within an enterprise IT department. The tool allows a replication latency threshold to be specified for each replication subscription, for the environment in question. The default latency threshold is 2 hours (120 minutes). If the subscriber data is older than the specified latency threshold, the tool raises an alert. The tool uses tracer tokens technology. Tracer tokens are inserted in publication databases at regular intervals. The system verifies that inserted tokens are moved to the subscription databases within the designated latency thresholds.
