Use Ctrl-F to find a specific code.

Some codes have a recommended action using PowerShell as well as the general action.

Links to other error code pages can be found at System Center 2012 – Virtual Machine Manager (VMM) Error Codes.

If you have additional information about an error, please add it to the "Additional Troubleshooting Information" column.

Code

Message

Recommended Action (in product)

Additional Troubleshooting Information

30100

The selected action could not be completed because the Virtual Machine Data Exchange information is only supported on Hyper-V.

Please specify a Hyper-V Virtual Machine.

30101

Virtual Machine %VMName; cannot be modified because Virtual Guest Services is not installed on the guest operating system or because the installed Virtual Guest Services does not support the Heartbeat and Data Exchange Integration Services in the current virtual machine status.

Verify the virtual machine is in a Running status, the Virtual Guest Services is installed, and the Heartbeat and Data Exchange Integration Services are enabled.

30102

Could not operate on virtual machine %VMName; Data Exchange Integration Services.

Ensure that the virtual machine exists and is in a Running status, the Virtual Guest Services is installed, and the Heartbeat and Data Exchange Integration Services are enabled and then try the operation again.

30103

The specified Data Exchange key name %PropertyName; does not exist on the virtual machine %VMName;.

Please specify a different Data Exchange key name and then try the operation again.

30104

The specified Data Exchange key and value pairs are not the proper length or data type.

Please specify a different Data Exchange key name and then try the operation again.

Specify a different Data Exchange set of key and value pairs, and then try the operation again.

30105

The specified Data Exchange key (%PropertyName;) already exists as a Guest Data Exchange key and should not be recreated as a Host Data Exchange key.

Specify a different Data Exchange key, and then try the operation again.
