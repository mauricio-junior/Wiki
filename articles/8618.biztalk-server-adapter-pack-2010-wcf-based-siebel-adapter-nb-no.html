Introduksjon

Den WCF baserte Siebel Adapter kan brukes til å koble BizTalk applikasjoner til Siebel .WCF baserte Siebel Adapter er en del av BizTalk Server Adapter Pack 2010. For å arbeide med adapteren trenger du å installere BizTalk Server 2010 Adapter Pack før du installerer dette adaptert (Adapter Pack er tilgjengelig gjennom BizTalk Server 2010 medium f.eks DVD / ISO). Denne artikkelen vil gi dine nødvendige ressurs lenkene for å finne informasjon rundt BizTalk WCF baserte Siebel Adapter, hvordan jobbe med det og så videre.
