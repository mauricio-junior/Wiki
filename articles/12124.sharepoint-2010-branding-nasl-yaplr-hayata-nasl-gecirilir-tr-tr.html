  • SharePoint Designer 2010’ü indirdikten sonra executabale'ını kullanarak yükleyebilirsiniz ( RUN’e basınız )

  • EULA okuyunuz (!) ve sonrasında continue

    • Ve yükleme işlemi başlar


    3 Basic Designs

    MasterPage, custom page ve benzeri “page” 'leri değiştirmek istersek iki tane feature’un aktif olması gerek.

    • · “Site Actions” 'a basınız
    • · “Site Collection Features” 'ına basınız
    • · “SharePoint Server Publishing Infrastructure” 'ın aktifleştiriniz

    • · “Site Actions” 'basınız
    • · “Manage site features” 'ına basınız
    • · “SharePoint Server Publishing” 'ını aktifleştiriniz.


    3.1 SharePoint Designer kullanim

    • SharePoint 2010’u açınız ve web sitenize giriniz.




    • Hiçbir zaman MasterPage'inizde direkt olarak değişikler yapmayınız. Önce bir kopya yapın, ismini değiştirin ondan sonra üzerine değişiklikler yapabiliriz.


    3.2 Custom CSS

    • Style library’de yeni bir CSS yaratalım.



    • Sağ klik ve “Edit File in Advanced Mode”

    • Ve içerisine aşağıdaki alanları kopyalayınız: 
     #s4-bodyContainer {
     width: 1024px !important;
     margin-left:auto;
     margin-right:auto;
     }
     body #s4-ribbonrow {
     background-color:#865102;
     }
     body.v4master {
     overflow: visible;
     background-color:black;
     body #s4-ribbonrow {
     }
     .ms-cui-ribbonTopBars {
     width: 1024px !important;
     margin-left:auto;
     margin-right:auto;
     }
     .ms-cui-ribbonTopBars > div {
     border-bottom:1px solid transparent !important;
     }



     



    • SharePoint Designer’de CSS’in Design mode'u sayesinde içine yüklediğiniz ve kopyalamış olduğumuz MasterPage’e sağ klik ve “New from Master Page” 'e basınız.



     

    • Yeni bir isim veriniz.



     

    • Sonrasında “Set as default Master Page”



     

    • Ve lay-out’umuz değişmiş bir şekilde karşımıza çıkar..
