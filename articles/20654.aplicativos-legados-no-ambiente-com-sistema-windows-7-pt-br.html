Prezados,

Por trabalhar em órgão público, por vezes aparecem sistemas que ainda não foram migrados para executar na plataforma Windows 7, o presente texto vem lembrar de uma ferramenta já disponível pela Microsoft, que poderá auxiliar na solução deste tipo de problema: é o chamado Modo Windows XP no Windows 7.

A solução se baseia no processo de virtualização, utilizando o software virtual PC com a complementação de um outro arquivo (uma estação pré configurada).

No meu caso em particular, tive que contornar a plataforma Windows 7 para a utilização do aplicativo DCTF Gerencial, em suas versões 2.14 e 4.8 e do site FGTS para a PGFN ( https://webp.caixa.gov.br/PortalPgfn/Index.asp ).

Todo o procedimento para instalação do chamado XPMode pode ser encontrado no endereço disponibilizado pela Microsoft, portanto, não é necessário transcrever o mesmo: 

LINK: http://windows.microsoft.com/pt-br/windows7/install-and-use-windows-xp-mode-in-windows-7

Caso enfrentem problemas semelhantes, recomendo utilizar esta solução.
