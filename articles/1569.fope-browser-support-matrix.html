Forefront Online Protection for Exchange Browser Support Level Definitions
Support Level
Definition
TIER 1
All advanced features should work well, full testing is done, and bugs fixed/triaged at par with any other feature. All bugs will be triaged and fixed.  There may be some cases where certain functionality (e.g. ActiveX controls) is only available only via a specific browser.
TIER 2
Tier2 support means all functionality should be accessible, but only a sanity test is required and we will rely on inherent compatibility offered by the browser.  However major bugs will be triaged and fixed.
TIER 3
Tier3 implies no support at all.  This tier encompasses all other browsers not listed in the current matrix.
