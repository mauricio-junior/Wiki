Resumo

Após instalação do Java, o arquivo “javaw” que costuma também ficar na pasta C:\Windows\System32 ou C:\windows\syswow64  em alguns casos, não pode mais ser encontrado acontece o erro abaixo:

 “Ocorreu erro 2 do Windows ao carregar Java VM”


Detalhes

Para solucionar esse problema siga os passos abaixo:

Instale a versão 32 bits(x86) mais atual do Java.

Va no Windows Explorer e vá até o caminho abaixo:

c:\arquivos de programas (x86)\java\jre1.8.0_65\bin  veja qual sua versão.

Copie o arquivo Javaw.exe

Depois vá até a pasta c:\windows\system32 e cole o arquivo nesta pasta, também cole na pastac:\windows\syswow64

Finalizando reinicie seu computador e teste novamente o Java.
