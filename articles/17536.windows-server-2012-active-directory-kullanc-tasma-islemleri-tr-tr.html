Merhabalar. Bu makalemizde, Windows Server 2012 Active Directory Domain yapısı içerisinde kullanıcı taşıma işlemlerinden bahsediyor olacağız. Bundan bir önceki makalemizde, yine Windows Server 2012 işletim sistemi üzerinde kullanıcı oluşturma işlemlerinden bahsetmiştik. Konu ile ilgili makaleyi incelemek isteyenler aşağıdaki adresten bilgi alabilirler.

http://www.yavuztasci.com/windows-server-2012-active-directory-zerinde-kullanici-olusturma/

Gerekli hatırlatmaları da yapıktan sonra, Active Directory üzeri n de kullanıcı taşıma işlemlerine geçiş yapabiliriz. Active Directory domain’i içerisinde kullanıcı taşıma işlemini gerçekleştirmek için, Active Directory Users And Computers yönetim konsolunu açıyoruz.

Ben, test amaçlı olarak Active Directory domain’im içerisinde  IT isminde bir adet Organizational Group ve bu grup içerisine de yine User1 ve User2 isminde iki adet kullanıcı oluşturdum. Oluşturmuş olduğum bu kullanıcıları, Active Directory içerisinde bulunan Users klasörü içerisine taşıyacağım.  Kullanıcı taşıma işlemi için, IT isimli klasörü içerisinde bulunan kullanıcılarımı seçip sağ tuş yapıyorum ve açılan menü içerisindenMove seçeneğini seçiyorum.

Move seçeneğinin ardından, karşımıza yukarıdaki şekil üzerinde de görmüş olduğunuz gibi bir pencere çıkacaktır. Bu pencere içerisinde,  kullanıcılarımızın hangi klasöre move edilmesini istiyor isek, o klasörü seçmemiz gerekmekte. Ben, kullanıcılarımın Users klasörü içerisine taşınmasını istediğim için, pencere içerisinden Users klasörünü seçiyorum ve OK tuşuna basıyorum.

OK işleminin akabinde, yukarıdaki şekil üzerinde de görüldüğü gibi  IT klasörümüzün içerisindeki kullanıcılarımızın taşınmış olduklarını görmekteyiz.  Kullanıcı taşıma işleminin sağlıklı bir şekilde tamamlanıp tamamlanmadığını test etmek için Users klasörünü kontrol ediyoruz.

Ve Users klasörü içerisine geri dönüp baktığımız da, User1 ve User2 isimli kullanıcılarımızın artık Usersklasörü içerisinde, yer aldıklarını görüyoruz.

Evet, bu makalemizde, Windows Server 2012 Active Directory Domain’i içerisinde, kullanıcı taşıma işlemlerinden bahsettik. Bir sonraki makalemde görüşmek üzere hoşcakalın.
