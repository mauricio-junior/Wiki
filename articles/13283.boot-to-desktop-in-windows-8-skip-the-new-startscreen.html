    Boot to Desktop in Windows 8 (Skip the New Startscreen)

    In just a few steps, you can boot directly to the desktop in Windows 8 without having to click the Desktop Tile all the time.

    1. Install Skip Metro Suite (http://winaero.com/comment.php?comment.news.103)

      <" " src="http://social.technet.microsoft.com/wiki/resized-image.ashx/__size/300x400/__key/communityserver-wikis-components-files/00-00-00-00-05/8054.skipmetro_5F00_main.jpg" style="border-width:0px;border-style:solid;" />
    2. select the check box "Skip Start Screen"
    3. click on "save settings"

    The next time the user logs into Windows, the tools enumerates all apps on the Start screen and virtually clicks the desktop tile to go directly to the desktop.


           

    If you have updated to Windows 8.1, you don't need 3rd Party Tools any longer. Go to the Desktop and make a right click on the taskbar and select "properties". Now click into the "Navigation" tab. Now set the check box at "When I sign in or close all apps on a screen, go to the desktop instead of Start."

     

           

    When you now start Windows 8.1, the new start Screen will be skipped.
