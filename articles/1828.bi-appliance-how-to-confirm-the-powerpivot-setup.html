The following section describes how to confirm the PowerPivot setup on the Microsoft Business Intelligence (BI) Appliance.

To confirm the PowerPivot Setup

  1. Download the Microsoft PowerPivot for Excel 2010 Samples.
  2. If you do not have PowerPivot for Excel installed, install the Excel 2010 trial and PowerPivot for Excel 2010 add-in from www.powerpivot.com.
  3. Go PowerPivot Gallery on the default Web site (http://appliancename).
  4. Go to Documents and Upload Documents, and then upload the LCA BI-Financial Report Usage.xlsx file.
  5. Perform the following tests:
    1. Make sure you can see the thumbnails for the document.
    2. Open the LCA BI-Financial Report Usage.xlsx file.
    3. Click the slicers within the document to make sure your model works (for example, the number changes).
