Salam Dostlar, növbəti GPO məqaləmizlə qarşınızdayıq. Bu məqalədə GPO ilə Windows 8 üzərində gələn Store(mağaza) və Microsoft Account`u ilə əməliyyət sisteminə daxil olmağı qadağan edəcəy-fragment-footer">

  Bunun üçün mütləq Windows Server 2012 üzərində Features`dən “Desktop Experience” aktiv edilməlidir. Əks halda Store aid policy`ni görmək mümkün olmayacaq. Əgər, özünüzdə yoxlamaq istəsəniz. Həmin policy üçün “Desktop Experience” aktiv etməmişdən əvvəl nəzər salın.

Aktivəşdirdikdən sonra serveri yenidən başladırıq. Daha sonra isə bu GPO`nu tətbiq etmək istədiyiniz istifadəçilər yerləşən Organizational Unit`ə yeni GPO tətbiq edirik. Həmin GPO`nu açmaq üçün “EDİT” düyməsinə sıxırıq və aşağıdakı yerə gəlirik: User Configuration\Policies\Administrative Templates\Windows Components\Store

Sağ tərəfdə yalnız bir ədəd policy`miz var. Bunun üzərinə vurub, açırıq.

Sonra şəkildən də gördüyünüz kimi bunu aktiv edirik. Nəzərinizə çatdırmaq istəyirəm ki, bu policy`ni yalnız Windows Server 2012, Windows 8 və Windows RT`yə tətbiq etmək olar. Group Policy`ni aktiv etdikdən sonra isə yeni istifadəçi üzərində Store`a daxil olmağa cəhd edək. Görək nəticə necə olacaq?

Şəkildən də gördüyünüz kimi nəticə tam olaraq bizim istədiyimiz kimi oldu. Artıq istifadəçilər Windows 8`in bu funksiyasından istifadə edə bilməyəcəklər. İndi isə məqalənin əvvəlində bəhs etdiyimiz Microsoft Account User mövzusuna qayıdaq. Bildyiniz kimi hər-hansı Microsoft Account`u ilə Windows 8`ə daxil olmaq mümkündür. Bunu aşağıdakı şəkildə də görmək olar.

Amma biz istəyirik ki, şirkətimiz daxilində yerləşən bütün kompüterləri istifadəçilər şirkət siyasətinə uyğun olaraq istifadə etsinlər. Bunun üçün yenə GPO`dan istifadə edəcəyik. Computer Configuration\Policies\Windows Settings\Security Settings\Local Policies\Security Options Yuxarıdakı yerə gəlib, burada sağ tərəfdə yerləşən “Accounts:Block Microsoft Accounts” policy`ni seçirik.

Bu policy`ni aktiv etmək üçün üzərinə iki dəfə sıxırıq və aşağıdakı pəncərə gəlir.

Buradan isə 3-cü seçim olan “Users can`t add or log on with Microsoft accounts” seçirik. Bu policy`ni də tətbiq etdikdən sonra yenə istifadəçi kompüterə üzərinə gəlib, tətbiq olunub, olunmadığını təsdiqləyək.

Gördüyünüz kimi policy`miz tətbiq olunub.Artıq istifadəçilər nə Store`a, nə də Microsoft Account ilə daxil ola biləcəklər. Amma bir şeyi nəzərinizə çatdırmaq istəyirəm ki, birinci tətbiq etdiyimiz policy diqqət etdinizsə, User(istifadəçi) səviyyəli idi. Ancq ikinci tətbiq etdiyimiz policy isə Computer(kompüter) səviyyəlidir. Bunlara diqqət etmək lazımdır. Əgər, ikinci policy`ni istifadəçilər yerləşən Organizational Unit`ə tətbiq etsək, bu zaman işləməyəcək. Ümid edirəm ki, yararlı məqalə oldu. Növbəti məqalələrdə görüşənə qədər…
