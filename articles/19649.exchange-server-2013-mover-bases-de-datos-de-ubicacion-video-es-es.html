Exchange Server 2013 almacena la información de los buzones en bases de datos lo que puede presentar problemas en un futuro si estas tienen un crecimiento descontrolado pudiendo incluso generar problemas de estabilidad general en el sistema que alberga estas.

Mover una base de datos puedo tener como objetivo no solo ampliar el espacio de almacenamiento de la unidad que la alberga, sino también., aumentar el rendimiento de esta por ejemplo, cambiando de tecnología y cambiando de discos IDE a SATA o de SATA a discos sólidos SSD lo que hace una opción más que interesante el trasladar las bases de datos de Exchange Server 2013 de una unidad a otr nueva y superior en prestaciones.

En este vídeo verás el proceso completo, el cual no entraña gran complejidad pero que como se ha tratado de transmitir en las líneas anteriores, puede suponer una gran mejora tanto de rendimiento como de almacenamiento para las organizaciones.
