.NET Framework Overview (es-ES)

  • Aplicar habilidades comunes a través de una variedad de tareas de programación, tipos de aplicaciones y dispositivos
  • Integración con otras herramientas y tecnologías para construir la solución correcta con menos trabajo
  • Crear aplicaciones atractivas más rápido

El Framework de .NET de Microsoft es un entorno de software que incluye una amplia biblioteca de código para solucionar problemas comunes de programación y una máquina virtual que administra la ejecución de programas.

El Framework de .NET es:

  • Entorno en tiempo de ejecución de lenguaje común - proporciona una capa de abstracción sobre el sistema operativo
  • Librerias de clase base – código preconstruido para tareas de programación comunes de bajo nivel.
  • Frameworks de desarrollo y tecnologías: Soluciones para grandes tareas de programación reutilizables y personalizables.

Las librerias de clase base del framework proporcionan una amplia gama de funciones que incluyen la interfaz de usuario, el acceso a datos, la conectividad a bases de datos, criptografía, desarrollo de aplicaciones web, algoritmos numéricos y comunicaciones de red. La librería de clases es utilizada por los programadores, que la combinan con su código para producir aplicaciones.

La versión 3.0 del Framewor de .NET se incluye con Windows Server 2008 y Windows Vista. La versión 3.5 se incluye con Windows 7, y también puede instalarse en Windows XP y la familia de sistemas operativos de Windows Server 2003.

La versión 4.0 fue lanzada en abril de 2010, junto con el lanzamiento de Visual Studio 2010. La versión 4.0 del framework de .NET puede instalarse desde Windows XP a Windows 7, con la excepción de las ediciones de Windows Server 2008 Core.
