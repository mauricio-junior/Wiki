Consultando Informações de Versão

Obtenha a atual versão instalada do SQL Server:

SELECT @@VERSION AS Version

ou se preferir:  

SELECT SERVERPROPERTY('Edition') AS Edition, SERVERPROPERTY('ProductVersion') AS ProductVersion, 
SERVERPROPERTY('ProductLevel') AS ProductLevel

O resultado será mostrado assim: "9.00.3054.00, SP2, Developer Edition."
