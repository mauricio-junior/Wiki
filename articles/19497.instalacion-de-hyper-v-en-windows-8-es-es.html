Hyper-V ha sido siempre una función de servidor de Windows en Windows Server 2008, Windows Server 2008 R2, Windows Server 2012, o como una instalación Stand Alone con Microsoft Hyper-V Server. Sin embargo, plataforma de Hyper-V ahora está incluida en el sistema operativo de Windows 8, así que son las herramientas de gestión.


Windows 8 incluye como una plataforma de virtualización Hyper-V, pero puesto que no todo el mundo utilizará esta función no está habilitada de forma predeterminada. Aquí es cómo habilitarlo en tu PC con Windows 8.

Cliente Hyper-V en Windows 8 es compatible con sistemas operativos de 32 bits y 64 bits. Si usted está instalando característica Hyper-V en un cliente Windows 8 de 32 bits, entonces usted sólo será capaz de instalar las herramientas de administración de Hyper-V, donde usted será capaz de gestionar de forma remota maquinas virtuales instaladas en servidores virtuales remotos.

Instalación de Hyper-V en un cliente de 64 bits le da control total para crear y administrar máquinas virtuales a través de Hyper-V Manager que sean creadas localmente o en servidores remotos.

Cliente Hyper-V soporta múltiples formas de almacenamiento de información, incluidas las unidades flash USB, IDE, iSCSI y SMB. Además, el cliente Hyper-V se aprovecha de las mejoras de seguridad y gestión de Windows 8, como el cifrado de unidad BitLocker, arranque seguro y así sucesivamente. Puede instalar cliente Hyper-V en cada dispositivo de 64 bits que tiene segundo nivel Dirección traducción (listón) habilitado. PCs más modernos tienen soporte para el listón. Soporta hasta 12 monitores, mayoría de los dispositivos USB, redes inalámbricas y dormir y Estados de hibernación. También ofrece una experiencia plena fidelidad, incluyendo 3D, audio, multi-touch y así sucesivamente.

Para instalar el cliente de Hyper-V en Windows 8, siga los pasos siguientes:

  1. Virtualización tiene que ser apoyado por su equipo y habilitado en la BIOS, entonces reinicie su equipo, entrar en la BIOS y asegúrese de que están habilitadas las opciones de virtualización.

  2. Desde la pantalla de Inicio Windows 8, escriba Panel de control


  3. Haga clic en Panel de control

  4. Desde el Panel de Control, haga clic en Programas y características


  5. Desde el panel del lado izquierdo de la la ventana programas y características , haga clic en características de Windows activar o desactivar.

  6. Cuando se abren las Características de Windows , seleccione la opción Hyper-V . Haga clic en Aceptar.

    Las dos opciones secundarias son Herramientas de administración de Hyper-V y Plataforma de Hyper-V



    Nota: Si está instalando característica de Hyper-V en un 32 bits cliente Windows 8, luego sólo serás capaz de instalar las herramientas de gestión, y la plataforma Hyper-V será atenuada como se muestra en la siguiente instantánea. Esto es verdadero si la virtualización no es admite o no habilitado desde el BIOS.

     

  7. Se iniciará la instalación de funciones seleccionadas, los binarios de Hyper-V ahora se agregarán a la instalación de Windows.

  8. Una vez que se habilita Hyper-V, totalmente debe cerrar y reiniciar el equipo para completar la instalación. Al reiniciar, usted será capaz de crear y gestionar máquinas virtuales a través de Hyper-V Manager o el módulo de Hyper-V para Windows PowerShell. También puede utilizar la conexión de la máquina Virtual para conectarse remotamente a VMs

    Para habilitar el cliente Hyper-V utilizando Windows PowerShell:

    En la línea de comandos de Windows PowerShell, escriba lo siguiente:
    Enable-WindowsOptionalFeature-en línea-nombredefunción Microsoft-Hyper-V



  9. El administrador de Hyper-V y la herramienta conexión de máquinas virtuales Hyper-V se ubicará en la pantalla de inicio

  10. Con el administrador de Hyper-V, usted será capaz de crear y gestionar máquinas virtuales, usted puede también ser capaz de conectarse remotamente a servidores Hyper-V




    Si usted es dueño de un cliente de 32 bits de Windows 8, entonces la plataforma Hyper-V no será instalado, por lo tanto, usted no será capaz de crear VMs. Sólo las herramientas de administración se instalará donde usted será capaz de conectarse a un servidor remoto Virtual.



Resumen

Hyper-V es ahora parte de Windows 8, sin embargo, no está habilitada de forma predeterminada. Una vez instalado, puede utilizar para crear y administrar las máquinas virtuales Hyper-V manager.
