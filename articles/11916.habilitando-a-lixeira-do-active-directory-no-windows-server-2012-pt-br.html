Habilitando a Lixeira do Active Directory no Windows Server 2 class="navigation-list-footer">

 

Introdução:

No Windows Server 2008 R2, recebemos uma nova funcionalidade no  AD DS, que era a lixeira do Active Directory, essa lixeira tem um papel importante dentro da estrutura e também para um administrador.

Mas no Windows Server 2008 R2 a lixeira era um pouco complicada de ser habilitada e de ser gerenciada, pois os passos para habilitar era via linha de comando com Enable-ADOptionalFeature e depois para restaurar era também um longo passo complicado, dependia de software de terceiros ou (Como restaurar os objetos do AD).

 

Aplica-se:

Agora vamos falar de algo novo como o Windows Server 2012.

 
Processo passo a passo de como Habilitar a lixeira:

No Windows Server 2012, o processo para ativar a Lixeira do AD DS é muito fácil e prático, apenas com 1 clique esse processo já é realizado, segue o procedimento:

 

 

 

 

Introduzido no Windows Small business o Active Directory Administrative Center ganhou muita notoriedade e diversas mudanças aconteceram para a versão do Windows Server 2012, também disponível no Windows Server 2008 R2, mas não muito usado por muitos, ele agora é responsável por uma série de configurações que facilitam a  vida de um administrador:

  • Com ele podemos alterar os níveis funcionais do Domínio e da Floresta
  • Configuramos o Dynamic Access Control
  • Configuramos templates de Senhas com o Password Setting Containers

E muitas outras opções de configurações. 

 

Mas nesse poso irei mostrar como Habilitar a Lixeira do AD DS para:

 

Recuperar qualquer Objeto deletado do AD DS.

 

  • Abre Gerenciador do Servidor
  • Abre Tools e depois o Active Directory Administrative Center

 

 

 

 

Após Abrir o Administrative Center, temos grifado em amarelo o domínio e também tem em verde o Enable Recycle Bin, veja que ficou muito mais fácil e agora sem linha de comando.

 

Apenas de um clique em Enable Recycle Bin e logo irá receber a mensagem abaixo.

 

 

 

 

Apenas confirme, detalhe, veja que confirmando não será mais possível desabilitar.

 

 

 

 

 

 

Após  confirmado, percebe que o Enable Recycle Bin está fosco (grifei em azul para ajudar), isso indica que está habilitado a lixeira para seu domínio.

 

Agora os objetos do AD deletados, podem ser recuperados num prazo de 180 dias.
