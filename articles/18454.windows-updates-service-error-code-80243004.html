Windows Updates Service Error Code 80243004

Table of Contents

  • Applies to
  • Error


Applies to


Windows Update Services, Windows Server 2008, Windows Server 2008R2

Error


Windows Update Services run for a long time and return error code 80243004.

Overview


That error is related to the fact that the services can’t display the “baloon” tip in the icon tray location.

For reference;  it could look that way

 

Resolve

Check the icon tray area and click to remove any empty field. Retry Windows Update after.
