SharePoint 2013: Modify Navigation Using Drag and Drop

SharePoint 2013 has a new feature : the ability to modify the navigation using Drag & Drop directly into the left and top menus.

I recommand you read

And you can drag documents / libraries / lists / list items in the menu to insert a link to the corresponding element.

You can also create levels, example:

  • First level as "lg15_Sp_IT .... "
  • Second level as "App_Offline."

Note that this feature does not work for files or for multiple moving documents.
