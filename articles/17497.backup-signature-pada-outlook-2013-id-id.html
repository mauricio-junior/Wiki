Mungkin Anda telah membuat banyak signature pada Outlook 2013. Jika suatu saat Anda melakukan instalasi ulang Windows dan Outlook 2013, signature ini dipastikan akan hilang dan Anda harus membuatnya kembali satu persatu. Sebagai solusinya agar Anda tidak perlu membuatnya lagi, maka Anda dapat membackup signature tersebut. Caranya sebagai berikut:

Jalankan Windows Explorer dan tampilkan kotak dialog Folder Options. Pada tab View, pastikan seting Show hidden files, folders, and drives telah diaktifkan.

Berikutnya masuklah ke lokasi berikut ini pada Windows Explorer:

C:\Users\NAMAUSER\AppData\Roaming\Microsoft\

Catatan: NAMAUSER adalah User Account yang Anda gunakan untuk login kedalam Windows dan yang Anda gunakan juga untuk mengakses aplikasi Outlook 2013.

Pada lokasi tersebut Anda dapat melihat folder yang bernama Signatures. Folder inilah yang menyimpan semua signature yang Anda buat pada Outlook 2013. Anda tinggal menyalin / membackup folder tersebut ke lokasi yang lain.

Dengan demikian, jika suatu saat Anda melakukan instalasi ulang Windows maupun Outlook 2013, maka Anda tinggal menyalin folder tersebut ke lokasi yang sama dan semua signature Anda pun dapat digunakan kembali tanpa harus membuatnya ulang satu persatu.
