The filters and sorts can be found below the search bar on the Forums 'home' page:

 

The Filter drop down has the following options:

 
These options can be put into three categories:

Thread state (only select one): 
  • All threads - This is the default filter. All threads are displayed regardless of their state (except deleted).
  • Answered - Only threads which have been marked as answered will be displayed. This is a great filter if you are looking for an answer. 
  • Unanswered - Only threads which do not contain a post marked proposed answer or answer will be displayed. This is a great filter if you are wishing to rack up your points.
  • Proposed answers -  
  • General discussion -

Thread attributes (can select multiple):

  • No replies - All threads which do not have a reply will be displayed. This is also great if you are wishing to rack up your points!
  • Helpful - When a post obtains 5 or more votes it will be considered 'helpful.' This filter will only display these helpful posts. This is another great filter if you are looking for a quality answer.
  • Has code - This filter only displays posts that contain code.

Language scope

  • All languages - This filter will bring you to a larger scope of all languages. This is designed to help bilingual users get out of the current 'language scope.' This feature is recommended to be used with search (see How to find content?) and/or scoping to content specific to the languages in which you speak (see How to customize your view?).

There are also several additional filter options if you have specific privileges to a forum such as being a moderator or owner. These additional filters include:

  • Abusive - Displays all threads which have been flagged as abusive. This filter is designed to allow moderators/owners to either un-mark posts if they have been falsely marked or to delete abusive posts. 
  • Deleted - Displays all deleted threads

The Sort drop down has the following options:

 

These options are defined as:

  • Most recent post - This is the default sort. This option sorts threads by the latest post or reply (newest messages on Forums).
  • Most recent thread - Sorts threads by their "newness." Threads will be sorted by when they were first posted.  
  • Total votes - Sorts threads by the highest amount of votes first.
  • Total replies - Sorts threads by the number of replies
