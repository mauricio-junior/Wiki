Disk Cleanup removes unneeded files, such as files that you downloaded to install programs, temporary Internet files, files in the Recycle Bin, offline Web pages, hibernation files, setup log files, temporary files, and thumbnails (Windows keeps a copy of all of your picture, video, and document thumbnails so they can be displayed quickly when you open a folder. If you delete these thumbnails, they will be automatically recreated as needed).
