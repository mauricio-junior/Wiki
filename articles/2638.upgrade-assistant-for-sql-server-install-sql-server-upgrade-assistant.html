3.3 Installing SQL Server Upgrade Assistant

Use the procedure below to install SQL Server Upgrade Assistant:

Ensure that your computer meets the system requirements.

  1. Double-click UAFSSetup.exe to launch the installation wizard.
  2. Review the License Agreement, click I accept the terms in the license agreement to select it, and then click Next.
  3. On the Destination Folder page of the installation wizard, you can click Change to specify the folder to install the Upgrade Assistant. If you wish to install in the default folder, simply click Next to continue.
  4. Click Install to begin the installation process.
Click Finish to close the wizard. You can launch Upgrade Assistant from the Scalability Experts programs group in the Start-Programs menu.
