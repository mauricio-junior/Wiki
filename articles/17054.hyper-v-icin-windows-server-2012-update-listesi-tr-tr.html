KB 2770917: Windows 8 and Windows Server 2012 cumulative update: November 2012
http://support.microsoft.com/kb/2770917
Açıklama: Hyper-V VM’lerinde Passthrough disk kullanıyorsanız ve backup sırasında hata ile karşılaşıyorsanız bu update’i mutlaka kurmalısınız. Ayrıca Hypervisor’un olası çökme durumundaki hataları gidermek için kurulmalıdır.
Yayınlanma tarihi: 13/11/2012
İşletim sistemleri: Windows 8, Windows Server 2012

KB 2770440: Poor performance when you enable RemoteFX in Hyper-V on a Windows Server 2012-based computer that uses AMD processors
http://support.microsoft.com/kb/2770440
AMD İşlemcili sunucularda RemoteFX aktifleştirilmiş Hyper-V üzerinde düşük performans sorunu yaşanıyor ise bu hotfix’in yüklenmesi gereklidir.
Yayınlanma tarihi: 14/11/2013
İşletim sistemleri: Windows Server 2012

KB 2785638: You cannot use the Cisco SR-IOV network adapter on a virtual machine in Windows Server 2012
http://support.microsoft.com/kb/2785638
Windows Server 2012 üzerinde Cisco SR-IOV network adaptör kartını kullanamıyorsanız bu hotfix’in kurulması bu sorunu ortadan kaldıracaktır. Bu hotfix, VM restart edildiğinde  cisco network kartının VF (virtual function) özelliğinin aktif olmaması hatasını gidermektedir.
Yayınlanma tarihi: 9/1/2013
İşletim sistemleri: Windows Server 2012

KB 2813630: Virtual machine enters a paused state or a CSV volume goes offline when you try to create a backup of the virtual machine on a Windows Server 2012-based failover cluster 
http://support.microsoft.com/kb/2813630
Windows Server 2012 tabanlı failover cluster yapısında bir backup yaratıldığında VM’ler pause state durumuna geçiyor ya da CSV Volume offline’e düşüyor ise aşağıdaki hotfix bu hatayı giderecektir.
Yayınlanma tarihi: 13/2/2013
İşletim sistemleri: Windows Server 2012

Bu hata, SR-IOV network adaptörleri kullanan sunucularda bazen VM’in crash olmasına neden olmaktadır. Aşağıdaki hotfix bu hatanın giderilmesini sağlamaktadır.
http://support.microsoft.com/kb/2806542

KB 2807654: "0x000000C2" Stop error when you try to back up a data deduplication-enabled volume in Windows 8 or Windows Server 2012
Windows 8 ya da Windows Server 2012 üzerinde, data deduplication özelliği aktif olan disk sürücülerinde  backup işlemi yapmaya çalışıldığında yukarıda hatayı almamak için aşağıdaki hotfix’in kurulması gerekmektedir.
http://support.microsoft.com/kb/2807654
Yayınlanma tarihi: 13/3/2013
İşletim sistemleri: Windows 8 ve  Windows Server 2012
