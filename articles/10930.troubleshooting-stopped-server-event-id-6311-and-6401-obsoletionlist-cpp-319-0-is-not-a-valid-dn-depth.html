OVERVIEW / PURPOSE
Recently in support, we have been seeing a new issue arise on the Active Directory Management Agent, and the FIM Service Management Agent.  We wanted to get this documented to help those that may be experiencing the issue, and attempting to resolve the issue.

PROBLEM STATEMENT 
You are executing a Full Import (Stage Only) on either Active Directory Management Agent, or the FIM Service Management Agent , or a Full Import and Sync on a Flat File MA , and you receive a "stopped-server" error message.  You investigate the Application Event Log and discover a series of errors including Event ID 6311 and 6401.
