MSDN/TechNet Forums - How to Add, Delete, or Edit an Announcement

 

If you have adequate permission to add, delete, or edit announcements in a forum there are several ways to go about it:

1)  Select the "Announcement" button. The easiest way to add, delete, or edit an announcement is to be in the current scope of the forum (See How to View a Single Forum?). Once in the scope of your forum you can select the "Announcement" button (image displayed above). You will see the announcement action buttons in the Announcement drop down:



2) Hover over the forum name. If you are in the scope of the forum in which you wish to add an announcement you can simply hover over the forum name and select "Add announcement":



 
3) Use the add an announcement URL. The URL of adding an announcement takes the form: http://social.msdn.microsoft.com/Forums/en-US/[FORUM]/announce. If you know the short name of the forum, simply paste it into the URL and you will be on the page to add an announcement.
