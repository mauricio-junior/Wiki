By default, Windows Server 2012 is not very dev friendly. I suggest some changes. At a high-level, do the following:
  1. Rename the server to something short, easy to remember and easy to type. I usually go stellar with this, with names like moon, mars, and nova.
  2. Enable Remote Desktop
  3. Disable Windows firewall
  4. Disable IE Enhanced Security
  5. Configure Windows update
  6. Configure time zone settings
