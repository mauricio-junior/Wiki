Below are simple steps to be followed to include a folder to the library from external media-which may be a an external hard disk or a USB drive etc.:-

1.Connect the external media to the computer.

2.Click  the File Explorer button present in the taskbar.File Explorer (previously known as Windows Explorer) allows you to open, access, and rearrange your files and folders in Desktop view.



3.In the navigation pane click Computer and then navigate to the folder on your external media which you want to include.

4.Right click on the folder and select Include in library and finally select the library (may be Documents,Music,Pictures,Videos).
