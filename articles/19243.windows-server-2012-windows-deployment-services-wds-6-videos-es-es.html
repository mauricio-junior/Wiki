Windows Deployment Services o Servicios de Implementación de Windows (WDS) es

En este artículo dejaré una serie de 6 vídeos en los que se tratarán los siguientes puntos.

1. Instalación de WDS
2. Agregar, eliminar y deshabilitar imágenes a WDS
3. Crear grupos de imágenes
4. Crear una imagen de captura de Sistemas Operativos
5. Capturar un sistema operativo
6. Generar archivos de respuestas y asignarlos

Este primer vídeo trata sobre la instalación y configuración inicial de WDS.




En este segundo vídeo agregaremos varias imágenes al servicio WDS e iniciaremos un sistema conectado al servidor para ver como funciona el servicio y como se muestra del lado cliente. Una vez agregadas las imágenes el vídeo muestra como deshabilitar o eliminar estas.




En esta ocasión se muestra como crear diferentes grupos de imágenes de forma que podamos organizar estas para que sean más fácil de administrar.




WDS no solo permite desplegar imágenes, también nos permite generar una imagen capturando un sistema operativo de un sistema de la red, para ello es necesario crear una imagen especial, proceso que se detalla en el siguiente vídeo.




Una vez generada la imagen de captura solo falta proceder generar una imagen y transferirla al servidor, este vídeo detalla algunas de las tareas que deberemos de llevar a cabo como por ejemplo la ejecución de un SysPrep en el sistema que queremos capturar.




Por último y para completar esta serie de vídeos vamos a ver como podemos generar un archivo de respuestas y aplicarlo a las imágenes de instalación que tenemos. Estos archivos de respuestas nos permiten automatizar algunas pasos del proceso de instalación agilizando en gran medida esta.
