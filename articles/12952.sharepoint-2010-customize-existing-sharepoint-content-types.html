SharePoint 2010: Customize existing SharePoint Content Types

 

 

 

Yes you create another list and add the columns you want, but sometimes you just need to add 1 or more columns in Announcements list.

 

 

 

Go to “Site Settings” > under Galleries click “Site Content Types”
You will see all Site Content Types, under “List Content Types” click Announcement, you can do the same with other content type.

 

 

 

Below are the existing columns for Announcements, click “Add from existing site columns”

 

 

 

Choose a column, I chose “Active” as exemple, you may develop a web part or event add a CQWP that displays Announcements and you want to filter by Active or not.

 

 

 

 Click ok, you will return to the columns, you will see “Active” column is added.

 

 

 

 Now if you click, Add new item, you will see your new column..
