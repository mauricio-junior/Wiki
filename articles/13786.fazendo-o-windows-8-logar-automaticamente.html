Introdução

Se Você não quiser digitar sua senha toda vez que o Windows 8 é iniciado, abaixo Nós vamos mostrar como fazer esse procedimento.

Configurando o Windows 8 para Logar Automaticamente

Pressione Windows + R e digite netplwiz e pressione Enter.


  
 
Isso abrirá a lista de todos os usuários em seu computador.
 
Agora selecione a conta de usuário da lista e em seguida desmarque a opção "Os usuários devem digitar um nome de usuário e senha para usar este computador" e clique em Aplicar.


  
 
Depois que clicar em aplicar uma caixa de dialogo abaixo será aberta , você completa a mesma colocando o usuário e senha a ficar salvo na hora de iniciar o Windows 8.
 

Agora reinicie sem micro e Isso fará com que o seu usuário inicie automaticamente o Windows.

Conclusão

Apesar de não ser uma boa dica na questão de segurança o procedimento acima pode ser muito útil a usuários domésticos, esse procedimento mostrou como Fazer o Windows 8 Logar automaticamente.
