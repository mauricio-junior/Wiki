Objetivo

Esse artigo tem como objetivo evitar a necessidade do uso do Windows Server para a instalação do SharePoint 3.0.

Um dos requisitos necessários para a instalação do SharePoint 3.0 é o uso de sistemas operacionais servidores, ou seja, Windows Server 2003 ou superiores. Sabendo que o Kernel utilizado no Windows 7 é o mesmo utilizado no Windows Server 2008 esse requisito obrigatório pode ser contornado e a instalação do SharePoint em ambientes clientes pode ser executada.

Pré-requisitos necessários

- Windows 7 Professional ou superior;

OBS: Home Basic não irá funcionar e Home Premium apresentará incompatibilidades devido a ausência do Editor de Política de Grupos (gpedit.msc)

- Ser administrador local ou possuir privilégios administrativos

Instruções

01) Instalação do WssVista 3.0

01a) Execute o instalador do WssVista 3.0.


01b) Caso o UAC (Controle de Conta de Usuário) esteja habilitado ele pedirá confirmação para continuar a instalação.


01c) Após a instalação ser concluída com sucesso surge a tela de confirmação.


01d) Após a instalação navegue até a pasta C:\Program Files (x86)\WssOnVista\Setup


02) Configurando o IIS

02a) Abra o Painel de Controle e vá em Programas e Recursos.


02b) Clique em Ativar ou Desativar Recursos do Windows.


 02c) Na janela vá em Serviços de Informações da Internet e marque todos os itens como exibidos abaixo:



 

 

03) Instalação do SharePoint 3.0

03a) Vá até a pasta onde está instalado o WssVista 3.0


 03b) Execute o SetupLauncher e aguarde a janela solicitando o instalador do SharePoint 3.0


 


03c) Escolha o modo Avançado.


03d) Escolha a opção Front-end da Web e clique em Instalar Agora.


03e) Aguarde a instalação ser completada.


03f) Após o término da instalação marque o campo Executar o Assistente de Configuração dos Produtos e Tecnologias do SharePoint agora.

 

03g) Após clicar em Fechar com o item marcado surge a seguinte janela.


03h) Ao clicar em Avançar surge a seguinte janela abaixo. Clique em Sim.


03i) Escolha a opção: Não, Desejo criar um novo farm de servidores.


03h) Ao clicar em Avançar surge a seguinte janela abaixo. Clique em Sim.


03i) Escolha a opção: Não, Desejo " src="http://social.technet.microsoft.com/wiki/resized-image.ashx/__size/442x362/__key/communityserver-wikis-components-files/00-00-00-00-05/4604.17.jpg" style="border-width:0px;border-style:solid;" />

03j) Nessa tela preencha os campos de acordo com o informado.


03k) Após inserir os dados corretamente surge a tela abaixo. Use a Autenticação NTLM.


03l) Certifique-se que os dados foram todos inseridos corretamente e clique em Avançar.


03m) Aguarde o processo ser concluído, pode demorar um pouco.


03n) Após a conclusão da configuração é exibida a seguinte tela.


03o) Após clicar em Concluir a tela do IIS com o SharePoint configurado é exibida.

Conclusão

Após esses procedimentos você será capaz de utilizar o SharePoint 3.0 em ambientes clientes Windows
