Merhaba ,

Herhangi bir sebepten dolayı ( Windows servislerinin başlatılmaya çalışırken “Access is Denied” hatası vermesi , Windows 7 ‘nin dhcp’den ip adresi alamaması vs… ) işletim sisteminin LGPO’sundan erişebileceğiniz local security policy’i default’a çevirmek isterseniz aşa">

Dikkat : Komut çalıştıktan sonra , default security policy gereği Administrator hesabını devre dışı bırakır ve password’ü kullanım dışı kalır. Bundan dolayı işletim sistemini yeniden başlatmadan önce bu duruma dikkat ediniz!!!
