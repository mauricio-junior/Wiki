We saw that with in some situations Disabling Malware Inspection used to fix the issue but in some case we have to disable Malware Inspection + NIS

After doing the investgation we have found that issue was with one of the Antimalware signature was released and that caused issue.

So to resolve this fix is :

Re enable Malware Inspection and then Install latest defnitions from Update center in TMG Console

We can check version in registry at this location HKLM \Software\Microsoft\fpc\EmpScanner\Versions\mpasdlta.vdm  and mpavdlta.vdm , It should be above 1.115.58.0
