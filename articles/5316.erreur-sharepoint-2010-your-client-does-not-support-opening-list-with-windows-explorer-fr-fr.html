J'ai reçu cette erreur en voulant ouvrir le Windows Explorer View dans une Document Library “Your client does not support opening list with windows explorer”

 

La solution:

Par default, “Web Client” service n'est pas installé sur Windows 2008R2, vous devez installez “Desktop Experience” via Server Manager > Features
