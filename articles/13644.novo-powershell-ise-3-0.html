Introdução



O novo powershell ISE 3.0 veio para facilitar o uso do powershell através de uma interface intuitiva, com diversas novas opções. Para quem não conhece a ferramenta, segue um tela abaixo:



Ele é basicamente composto de 3 painéis: 
  • Scripts - Nesse painel você pode colar seus scripts e pode executar por completo (F5) ou selecionar parte do código e executar somente o trecho selecionado (F8).
  • Comandos - Aqui você tem todos os comandos disponíveis, com ajuda em texto e também um facilitador de execução, que lhe pergunta os parâmetros que deseja executar o comando e já insere a linha pronta no painel do console.
  • Console - O console do powershell propriamente dito.
