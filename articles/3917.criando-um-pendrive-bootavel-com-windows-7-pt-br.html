 Introdução

 O Artigo abaixo mostra como criar um pendrive bootalvel com Windows 7, Lembrando que o método de instalação por USB é muito mais rápido.

 Nota: Utilizaremos o Windows 7 USB DVD Download Tool da propria Microsoft para fazer o processo.

 Criando um pendrive bootável

1.  Primeiramente acesso o site e baixe o arquivo: http://images2.store.microsoft.com/prod/clustera/framework/w7udt/1.0/uk/Windows7-USB-DVD-tool.exe

2.  Execute o arquivo.

3.  Agora no Utilitário clique em Next e install até finalizar.

4.  Agora clique no programa que foi baixado.

 

5.  Agora em “Browse”, selecione o arquivo isso do Windows 7 que você adquiriu legalmente junto a Microsoft e clique em Next.

 
 
6.  Escolha um tipo da mídia, USB divice.



 
7.  Coloque o Pen Drive, selecione-o no menu e clique Begin copying.


 
8.  Será criado o Pen Drive com o Windows 7.


 
9.  Seu Pen Drive bootável com o Windows 7 foi criado com sucesso!



 
10. Agora e só utilizar o seu pen drive esta pronto para fazer a instalação do Windows 7 em qualquer computador que faça boot via USB.
