Xoş History

Remote Server Administration Tools

a) İdarə edə biləcəyimiz Server Sistemləri

- Windows Server 2003, 2008, 2008 R2

b) RSAT dəstəklədiyi Əməliyyat Sistemləri (ƏS)

- Windows 7 Enterprise, Professional, Ultimate

1. Əməliyyat sisteminizə uyğun RSAT paketini yükləmək üçün aşağıdakı keçidlərdən yararlana bilərsiniz.

a) RSAT Paketini yüklə

b) RSAT vasitəsilə İdarə oluncaq rolların siyahısı

2. Paket çox sadə şəkildə yüklənir.

3. I Accept - Close

4. Yükləmə tamamlandıqdan sonra Start- Control Panel - Programs and Features - Turn Windows features on or off bölməsinə daxil oluruq.

5. Remote Server Administration Tools altında yerləşən rollaradan ehtiyac olanları seçərək yükləyirik.

6. Artıq Administrative Tools bölməsini açdıqda Windows 7 üzərindən idarə edəcəyimiz Server rollarının yükləndiyini görürük.

7. Misal üçün DNS xidməti yüklü olan Serverinizə qoşulmaq üçün DNS Manager üzərindən sağ düymə vasitəsilə Connect to DNS Server seçərək istədiyiniz DNS Serveri idarə edə bilərsiniz.

Digər rolları eyni qayda ilə yükləyərək idarəçəliyi Windows 7 üzərindəndə həyata keçirə bilərsiniz. Başqa bir məqalədə görüşmək arzusu ilə.
