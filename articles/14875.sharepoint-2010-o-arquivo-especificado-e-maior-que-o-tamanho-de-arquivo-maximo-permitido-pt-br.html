Introdução:

Bem, no SharePoint 2010 temos um recurso muito importante chamado sistema de cotas, essas cotas são de suma importância porque permitem aos administradores um eventual controle do tamanho dos arquivos trafegados em sites, bibliotecas e outros repositórios presentes no SharePoint. Esse controle é feito através do upload dos arquivos e refletido para o conjunto de sites criados dentro de uma estrutura raiz de um aplicativo web.

Por padrão o tamanho é 50 MB que pode ser fácilmente alterado para comportar a necessidade da empresa.

Referência Microsoft:

http://technet.microsoft.com/pt-br/library/cc263223(v=office.14).aspx

O que vale frizar nesse momento é que se você tentar realizar um upload de um determinado arquivo maior que o tamanho permitido você receberá a seguinte mensagem:

"O arquivo especificado é maior que o tamanho de arquivo máximo permitido".

Este problema ocorre porque o arquivo que está sendo carregado para a biblioteca de um site por exemplo é maior que o tamanho de 50 mb.

Exemplo:



Para corrigir este problema é necessário alterar o limite de cotas no servidor de aplicação onde o SharePoint 2010 encontra-se instalado.

Pré-requisitos:

  • Ter direitos de administrador ao servidor de aplicação onde o sharepoint encontra-se instalado.
  • Ter acesso a administração central do SharePoint 2010

Iniciando a correção:

Acesse o servidor de aplicação onde encontra-se instalado o SharePoint 2010.

Acesse a: Central Administration - "Administração Central" - Application Management - "Gerenciamento de Aplicativo"- Manage Web Application - "Gerenciamento de Web Application".

Exemplo:




Selecione o: Aplicativo Web - "Web Application" e clique em: Configurações Gerais - "General Settings".

 Exemplo:



Configure o tamanho máximo do arquivo de upload - "Maximum Upload Size" de 50 MB para o tamanho necessário a necessidade de sua empresa.

Exemplo:




Ao término clique no botão OK, retorne ao site onde apresentou e faça novamente o upload do arquivo.

Outras informações relacionadas aos limites e tamanhos máximo de arquivos também podem ser definidas utilizando o Microsoft Forefront.
