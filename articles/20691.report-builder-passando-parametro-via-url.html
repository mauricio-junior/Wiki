Quando temos relatórios construídos no Report Builder , e precisamos utiliza-los via web, onde seria necessário passar os valores dos parametros para que os dados sejam exibidos conforme as 

A sintaxe ficará da seguinte forma

http://<server>/ReportServer/Pages/ReportViewer.aspx?%2f<path>%2f<ReportName>&rs:Command=Render&<ParameterName>=<value>


Caso tenha mais de um parâmetro concatene-os utilizando esta forma &<ParameterName>=<value>
