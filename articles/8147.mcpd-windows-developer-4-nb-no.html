The MCPD: Windows Developer 4 er for utviklere som bygger rike klientapplikasjoner for "Windows Form"-plattformen ved hjelp av Microsoft. NET Framework 4.

Nødvendige eksamener 
  • Exam 70-511 - TS: Windows Applications Development with Microsoft .NET Framework 4
  • Exam 70-513 - TS: Windows Communication Development with Microsoft .NET Framework 4
  • Exam 70-516 - TS: Accessing Data with Microsoft .NET Framework 4
  • Exam 70-518 - PRO: Designing and Developing Windows Applications Using Microsoft .NET Framework 4
