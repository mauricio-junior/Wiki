Problem:

Windows 7 Basic users may  get the following message during watching movies in VLC Player:

The color scheme has been changed to Windows 7 Basic...

The screen goes black without any audio.

Resolution:

To fix the above problem try the following:

1] Update your graphic card driver. Visit the website of the graphic card manufacturer to download and install the latest driver.

2] i]  Open VLC media player. 

    ii]  Go to menu click on 'Tools'.
 
    iii] Click on 'Preferences' > 'Video' > 'Output'.

    iv]  Check the settings: it should be default,  if required you may  need to change settings to 'Windows GDI video output' > press 'Save' button.






3] Make sure that you are using the latest version of  VLC Player.
