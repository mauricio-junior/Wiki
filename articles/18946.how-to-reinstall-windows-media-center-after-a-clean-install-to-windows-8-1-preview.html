pan> Hi If you update your Windows 8 Pro with Windows Media Center to Windows 8.1 Preview with a Clean install then you will find that Windows Media Center (WMC) is no longer available and you need to reinstall WMC again.

Here are step by step instructions on how to:-

1.Open Search.

2.Enter add features in the search box, and then tap or click Settings.

3.     Tap or click Add features to Windows 8 and then do one of the following:

  • If you need to purchase a product key, tap or click I want to buy a product key online.

    Follow the steps to purchase and enter a product key.

  • If you already have a product key, tap or click I already have a product key.

4.Enter the exact product key which you used during WMC installation on Windows 8.Click Next.

5.Select the check box to accept the license terms, and then click Add features.

6.PC will restart automatically and Windows Media Center will be available for use.
