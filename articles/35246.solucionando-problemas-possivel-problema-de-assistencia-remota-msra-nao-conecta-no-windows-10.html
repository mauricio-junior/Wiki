Resumo

Se tudo estiver corretamente configurado para assistência remota e esse erro acontecer, pode ser devido a uma permissão que não é incluída nas configurações DCOM no Windows 10. 

Detalhes

Para solucionar primeiramente acesse as Configurações DCOM no Windows 10.

Digite o comando no iniciar executar dcomcnfg

Agora navegue até:

 

Componentes de Serviços – Meu Computador -  CONFIG DCOM

 

E procure pela caixa RASERVER clique com o botão direito do mouse na caixa e em propriedades.

 

Navegue até a aba segurança e set as permissões do Grupo ou usuário que vai fazer o acesso remoto nas opções marcadas abaixo.

 

Incluindo as permissões, tente novamente acessar o MSRA o problema será solucionado.
