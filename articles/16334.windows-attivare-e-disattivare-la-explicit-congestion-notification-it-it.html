La Explicit Congestion Notification (ECN) è una funzionalità presente in Windows a partire da Windows Vista e Windows Server 2008 che consente la notifica end-to-end della congestione di rete in modo da evitare la perdita di pacchetti di dati quando se ne stanno gestendo troppi insieme.
La ECN è un'estensione ai protocolli TCP e IP definita nell'RFC 3168; si tratta di una caratteristica opzionale, utilizzata solo quando è supportata da entrambi gli endpoint della connessione e gli stessi sono disposti ad utilizzarla.

Per impostazione predefinita, la funzionalità è disattivata; per attivarla, aprire la finestra del prompt dei comandi ed eseguire il comando

netsh int tcp set global ecncapability=enabled

Per disattivare la funzionalità eseguire il comando

netsh int tcp set global ecncapability=disabled
