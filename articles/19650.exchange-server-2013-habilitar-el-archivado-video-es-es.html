Exchange Server 2013 provee a los administradores de un mecanismo mediante el cual podemos conceder a los usuarios la posibilidad de almacenar ciertos mensajes no tan importantes pero que desean conservar en una base de datos alternativa, de esta forma podemos repartir la carga de almacenamiento en dos bases de datos de una forma totalmente transparente para el usuario final.

Esto puede ser una gran ventaja sobre todo en empresas que han actualizado anteriormente su infraestructura y que cuentan con hardware antiguo, que no ofrece el rendimiento que exigen las aplicaciones actuales pero que puede ser reciclado por ejemplo, para esta función.

Como vemos en el vídeo el proceso a llevar a cabo para habilitar esta función es realmente sencillo y bastará con crear una base de datos a la que asignaremos el archivado de los usuarios..
