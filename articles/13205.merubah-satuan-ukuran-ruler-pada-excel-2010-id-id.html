Merubah Satuan Ukuran Ruler pada Excel 2010 (id-ID)

Pada aplikasi Excel 2010, pada mode tampilan Page Layout, ukuran ruler default yang digunakan adalah ukuran Inch.

Mungkin saja Anda tidak terbiasa dengan satuan ukuran Inch tersebut maka Anda dapat merubahnya ke satuan ukuran lainnya dengan mudah. Caranya klik menu File -> Options untuk menampilkan jendela Excel Options. Kemudian pilih tab Advanced, dan di bagian Display -> Ruler units, pilihlah satuan ukuran yang Anda inginkan dan klik tombol OK untuk menutup jendela tersebut.

Hasilnya satuan ukuran pada ruler di mode tampilan Page Layout akan berubah mengikuti satuan ukuran yang telah Anda pilih sebelumnya.
