Out of the box it's not possible to copy permissions between site collections. You can copy permissions programmatically, as explained in http://mosswat.blogspot.com/2009/02/how-to-copy-permiss1">
http://social.msdn.microsoft.com/Forums/en/sharepointdevelopment/thread/25e54544-bf02-4861-af6d-a8a308c58b34.

However, it's considered a best practice to make a security group in Active Directory and add that security group to each of the site collections. Then all you need to do is assigning the group with same permission level.
