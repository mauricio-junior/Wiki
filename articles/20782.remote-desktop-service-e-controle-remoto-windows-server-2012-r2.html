​Nesse artigo vou mostrar como utilizar o "Controle Remoto" do Remote Desktop Server no Windows 2012 Server R2. No Windows 2012 Server muitos administradores sentiram a falta do "Controle Remoto" que existe no Windows 2008 Server, que foi removido na versão do Windows 2012 Server. Com a nova versão R2, a ferramenta foi reintroduzida e pode ser utilizada normalmente pelos administradores. Para ter essa opção disponível, na instalação do RDS, você precisa escolher a opção "Implantação de área de trabalho baseada em sessão", para que a opção "Sombra" apareça nas tarefas do RDS.
