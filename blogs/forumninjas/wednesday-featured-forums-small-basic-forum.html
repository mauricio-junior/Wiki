Lets learn first what is Microsoft Small Basic? Microsoft Small Basic is a programming language which is simplified version of BASIC programming language intended as an easy programming language for beginners. Yes you read it right "Easy" and "Beginner". As its name suggests a small and simple language that makes it possible to easily accessible, learn and teach to complete beginner about programming. The Environment is simple but rich in features, offering beginners several of the benefits like a rich set of Libraries help beginners learn by writing compelling and interesting programs.

The unique feature of the Small Basic language is that unlike other programming languages it consists 14 keywords and another 2 operators (And, Or), totaling 16 reserved words. A list of Small Basic keywords is shown below.

If, Then, Else, ElseIf, EndIf, For, To, Step, EndFor, While, EndWhile, Sub, EndSub, Goto, And, Or

Small Basic forum contains 4786 pages of questions and answers of great collaboration, where community members are discussing their issues and solution approach and learning from each other. This forum also has a monthly programming challenges containing compelling and interesting topics from various categories like below for beginner to learn in easy and interesting way.

  • Graphics Challenges
