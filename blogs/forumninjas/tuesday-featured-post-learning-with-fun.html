Hey Guys!

We are back with the Tuesday’s Featured Post where we discuss a Forum Post or a Forum Thread from the MSDN or TechNet Forums and then highlight the value that they added.

For this week theme I was thinking writing about a forum post or a thread that discusses about easier ways of learning programming languages. So, while browsing through the forum I came across an interesting and fun forum post, Challenge of the Month - July 2018 from Small Basic Forum. This forum post is about how you can start learning Small Basic programming and more more importantly having a fun when learning programming.

Be humble. Be teachable. Always keep learning.

This forum is a good example of how one can start learning programming not only Small Basic but also other programming languages because the theme is the same learn with fun. If we look closely the details that is written in the post about this thread, first we will see that this forum post contains a collection of interesting programming challenges that is used to learn Small Basic programming. If i quote from the post,

"These challenges are intended for people who are learning to program for the first time or for those returning to programming who want to start using Small Basic. Some will be easy, some will be hard - but they will all make you think, and more importantly be GREAT FUN!"

We also see that this post also encourages the community members to share if any problem or issue they faced during solving these challenges so that a discussion can start and member's can learn from each other. Also if anyone has an idea for future challenges they can share it here so that challenges can continue in next month.

Some of the programming challenges category from this post:

  • Graphics Challenges
  • Maths Challenge
  • Game Challenge
  • Turtle Challenge

With this forum post we can see that there are various ways of learning programming and one of them is with some interesting and fun programming challenges. I hope that this post helps and see you on the next one.

Thank You,
-Ninja Sabah
